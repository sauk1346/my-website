## Herramientas

- **Google Colab:** Plataforma gratuita de Google que permite ejecutar código Python en la nube usando notebooks Jupyter. Ofrece acceso a GPUs y TPUs sin costo, ideal para machine learning y análisis de datos. Se integra perfectamente con Google Drive y permite colaboración en tiempo real.
- **Kaggle:** Comunidad global de ciencia de datos que ofrece competencias de machine learning, datasets públicos y notebooks colaborativos. Incluye cursos gratuitos, un entorno de desarrollo en la nube con GPUs, y es perfecto para aprender, practicar y mostrar proyectos de data science.
- **Deepnote:** Plataforma colaborativa de notebooks diseñada específicamente para equipos de datos. Combina la funcionalidad de Jupyter con herramientas modernas de colaboración, control de versiones integrado, y capacidades de publicación. Soporta múltiples lenguajes y ofrece integraciones con bases de datos y servicios en la nube.
