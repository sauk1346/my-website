# Ciclos

Es un bucle que repite un bloque de código hasta que se cumple una condición. Hay dos tipos de ciclos en Python:

- Sentencias `for`
- Sentencias `while`

## 1. Sentencia For
Nos permite iterar **sobre una secuencia de elementos** como por ejemplo cadenas de texto, listas y diccionarios.

```python
for <var> in <iterable>:
    <statement(s)>
```

**Ejemplo 01**

```python
for i in range(1,11):
    print(i)
```

**Ejemplo 02**

```python
numbers = [0, 1, 2, 3, 4]

for num in numbers:
```

## 2. Sentencia While
Nos permite ejecutar un bloque de instrucciones mientras se cumpla la condición dada.

**Ejemplo 01:** Saludo

```python
want_greet = "S" # importante dar un valor inicial

while want_greet == "S":
    print("Hola que tal!")
    want_greet = input("¿Quiere otro saludo? [S/N]: ").upper()

print("Que tenga un buen día")
```

**Ejemplo 02**

```python
day = 0
week = ["Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado", "Domingo"]

while day < 7:
    print(f"Hoy es {week[day]}")
    day += 1
```
<TerminalOutput withBg>
{`Hoy es Lunes
Hoy es Martes
Hoy es Miércoles
Hoy es Jueves
Hoy es Viernes
Hoy es Sábado
Hoy es Domingo`}
</TerminalOutput>

## 3. Sentencias Break y Continue

- `break`: Podemos parar o romper la ejecución de un ciclo si así lo requerimos. Esta termina el ciclo por completo y continúa con la primera declaración que sigue al ciclo.
- `continue`: Terminamos la iteración actual y continuamos con la siguiente

Ambas sentencias pueden usarse en ciclos `for` y `while`

**Ejemplo 01:** uso de `break`

```python
animals = ["cat", "dog", "rabbit", "horse", "dolphin"]

for animal in animals:
    if "o" in animal:
        break
    print(animal)

print("Done!")
```
<TerminalOutput withBg>
{`cat
Done!`}
</TerminalOutput>

**Ejemplo 02:** uso de `continue`

```python
animals = ["cat", "dog", "rabbit", "horse", "dolphin"]

for animal in animals:
    if "a" in animal:
        continue
    print(animal)

print("Done!")
```
<TerminalOutput withBg>
{`dog
horse
dolphin
Done!`}
</TerminalOutput>

## 4. Sentencia else (en ciclos)

Con la sentencia `else` podemos ejecutar código si se recorrieron todos los elementos de nuestro ciclo. Si por alguna razón el código fue interrumpido por un `break`, esta porción de código no se ejecutará.

Se puede usar en ciclos `for` y `while`

**Ejemplo 01**

```python
animals = ["cat", "dog", "rabbit", "horse", "dolphin"]

for animal in animals:
    print(animal)
else:
    print("Done!")
```
<TerminalOutput withBg>
{`cat
dog
rabbit
horse
dolphin
Done!`}
</TerminalOutput>

**Ejemplo 02**

```python
animals = ["cat", "dog", "rabbit", "horse", "dolphin"]

for animal in animals:
    if "o" in animal:
        break
    print(animal)
else:
    print("Done!")
```
<TerminalOutput withBg>
{`cat`}
</TerminalOutput>

## 5. Ejercicio

Escribe un programa que imprima los números del 1 al 100.

- Si el número es múltiplo de 3 imprimir Fizz.
- Si el número es múltiplo de 5 imprimir Buzz.
- Si el número es múltiplo de 3 y 5 imprimir FizzBuzz

<Solution>
```python
for num in range(1,101):
    if num%3 == 0 and num%5 == 0:
        print(f"{num}: FizzBuzz")
    elif num%3 == 0:
        print(f"{num}: Fizz")
    elif num%5 == 0:
        print(f"{num}: Buzz")
    else:
        print(num)
```
</Solution>
