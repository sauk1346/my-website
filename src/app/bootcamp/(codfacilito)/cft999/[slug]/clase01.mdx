## 4. Extracción de información

- Análisis de documentos generales
- **Lectura:** OCR para texto impreso y escrito
- **Diseño:** extracción de texto y estructura
- **Documento general:** extracción de texto, estructura y pares clave-valor
- Modelos predefinidos para tipos de forumularios comunes
- Entrenamiento de modelos personalizados para sus propios formularios
- **Plantilla personalizada:** extracción de datos de diseños estáticos
- **Neural personalizado:** extracción de datos de documentos de tipo mixto
- **Composición personalizada:** colección de múltiples modelos asignados a un único modelo
