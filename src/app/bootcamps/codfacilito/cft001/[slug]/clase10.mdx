# Clase 10: ¿Qué es la visualización?

## Herramientas

### Altair
Altair es una biblioteca de visualización de datos estadísticos para Python que se basa en la gramática de gráficos de Vega-Lite. Fue desarrollada para crear visualizaciones declarativas de manera simple y expresiva.

**Ejemplo**

```python
import altair as alt
import pandas as pd

source = pd.DataFrame({
    'a': ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I'],
    'b': [28, 55, 43, 91, 81, 53, 19, 87, 52]
})

alt.Chart(source).mark_bar().encode(
    x='a',
    y='b'
)
```
