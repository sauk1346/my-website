## 4. Manipulación de Números y F-string

- `round(number,ndigits)`: redonde el número a n-decimales

```python
number = 3.66666
print(round(number,2))
```
<TerminalOutput withBg>
{`3.67`}
</TerminalOutput>

- `+=`, `-=`, `*=`, `/=` : permite abreviar operaciones sobre una misma variable

```python
a,b,c,d = 3, 3, 3, 3

a+=2
print(a)

b-=2
print(b)

c*=2
print(c)

d/=2
print(d)
```
<TerminalOutput withBg>
{`5
1
6
1.5`}
</TerminalOutput>

- **f-string:** permite agregar variables dentro de un string

```python
score = 0
height = 1.8
is_winning = True

print(f"Your score is = {score}, your height is {height}. You are winning")
```
<TerminalOutput withBg>
{`Your score is = 0, your height is 1.8. You are winning`}
</TerminalOutput>

## 5. Proyecto: Tip Calculator

```python
print("Welcome to the tip calculator!")
bill = float(input("What was the total bill? $"))
tip = int(input("How much tip would you like to give? 10, 12 or 15? "))
split = int(input("How many people to split the bill? "))

each_person = (bill + tip)/ split

print(f"Each person should pay: ${each_person}")
```
