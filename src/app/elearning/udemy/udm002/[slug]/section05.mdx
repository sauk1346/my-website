## Cómo almacenan Datos las Computadoras

- **CPU:** es el cerebro de la computadora que ejecuta las instrucciones y realiza los cálculos. Cuando necesita trabajar con datos, los solicita desde el almacenamiento.
- **RAM:** actúa como memoria temporal de trabajo rápido. Los datos y programas se cargan desde el disco de almacenamiento hacia la RAM para que la CPU pueda acceder a ellos de manera casi instantánea. Es volátil, por lo que pierde la información al apagar la computadora.
- **Disco de Almacenamiento (HDD o SSD):** guarda permanentemente los datos, programas y el sistema operativo. Es más lento que la RAM pero conserva la información incluso cuando se apaga el equipo.

### Flujo típico
Cuando abres un programa, se copia desde el disco de almacenamiento a la RAM. La CPU lee las instrucciones y datos desde la RAM, procesa la información y envía los resultados de vuelta. Si necesitas guardar algo, la CPU envía los datos desde la RAM al disco de almacenamiento para conservarlos permanentemente.

## Operaciones en Estructuras de Datos

- **Inserción:** Operación que agrega un nuevo elemento a la estructura de datos en una posición específica o según las reglas de la estructura.
- **Eliminación:** Operación que remueve un elemento existente de la estructura de datos, reorganizando los elementos restantes si es necesario.
- **Recorrido:** Operación que visita sistemáticamente todos los elementos de la estructura de datos en un orden específico para procesarlos o examinarlos.
- **Búsqueda:** Operación que localiza un elemento específico dentro de la estructura de datos, determinando si existe y en qué posición se encuentra.
- **Ordenamiento:** Operación que reorganiza los elementos de la estructura de datos siguiendo un criterio específico (ascendente, descendente, alfabético, etc.).
- **Acceso:** Operación que permite obtener o modificar el valor de un elemento específico en la estructura de datos, generalmente mediante su índice o clave.
