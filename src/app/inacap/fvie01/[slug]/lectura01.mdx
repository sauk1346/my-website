Técnica cualitativa. Permite el registro de información relativa a las experiencias de un usuario

## Consejos
Etapa de diagnóstico

1. Armar una pauta semi-estructurada para un dialogo fluido
2. Definir objetivos, temas y actores
3. Procurar un buen ambiente y comodidad para el entrevistado
4. Detectar inconsistencias (preguntar "por qué")
5. Estar atento al lenguaje no verbal
6. Evitar inducir las respuestas
