# AFO 2

Un faro construido al nivel del mar, tiene 91 metros de alto. Desde su cima, el ángulo de depresión a una boya mide 59°.

1. **Determine la distancia entre la base del faro y la boya.**

<Solution>

<SmartFigure
	src="/inacap/mvfm01/images/fig150.svg"
	alt="Representación de faro y boya"
	caption="Representación de faro y boya"
	size="large"
/>

$$
\begin{aligned}
    Distancia\;horizontal\;faro\;a\;boya = B
\end{aligned}
$$

Se tiene la siguiente relación

$$
\begin{aligned}
    \tan(59) = \frac{A}{B}\\[10pt]
    B = \frac{A}{\tan(59)}\\[15pt]
    B = \frac{91}{\tan(59)}
\end{aligned}
$$

<SmartFigure
	src="/inacap/mvfm01/images/fig151.png"
	alt="Cálculo distancia horizontal faro a boya"
	caption="Cálculo distancia horizontal faro a boya"
	size="large"
/>

$\therefore$ La distancia del faro hacia la boya es de `54.68` metros
</Solution>

2. **Determine la distancia entre la parte más alta del faro hasta la boya.**
<Solution>

<SmartFigure
	src="/inacap/mvfm01/images/fig150.svg"
	alt="Representación de faro y boya"
	caption="Representación de faro y boya"
	size="large"
/>

$$
\begin{aligned}
    Distancia\;más\;alta\;faro\;a\;boya = C
\end{aligned}
$$

Se tiene la siguiente relación

$$
\begin{aligned}
    \sin(59) = \frac{A}{C}\\[10pt]
    C = \frac{A}{\sin(59)}\\[15pt]
    C = \frac{91}{\sin(59)}
\end{aligned}
$$

<SmartFigure
	src="/inacap/mvfm01/images/fig152.png"
	alt="Cálculo distancia más alta faro a boya"
	caption="Cálculo distancia más alta faro a boya"
	size="large"
/>

$\therefore$ La distancia entre la parte más alta del faro hasta la boya es de `106.16` metros
</Solution>
