## Instrucción if()
- Es necesario respetar la indentación para las instrucciones dentro del `if()`.
- Si no está indentado, python asume que dichas instrucciones no pertenecen al `if()`

**Ejemplo 01:** 

- Semáforo.
- Si luz-verde imprimir 'Cruzar la calle'.
- Sino imprimir 'Esperar'

```python
semaforo = input("ingrese luz semáforo (rojo/verde): ")

if (semaforo == "verde"):
	print("Cruzar la calle")
else:
	print("Esperar")
```

**Ejemplo 02**: *if anidados*

- Ingresar compra
- Si es menor o igual a 100 imprimir "Pago en efectivo"
- Si es mayor a 100 y menor a 300 imprimir "Pago con tarjeta de débito"
- Sino, imprimir "Pago con tarjeta de crédito"

```python
compra = int ( input("Ingrese el valor de la compra: ") )

if (compra <=100):
	print("Pago en efectivo")
elif (compra > 100) and (compra < 300):
	print("Pago con tarjeta de débito")
else:
	print("Pago con tarjeta de crédito")
```

## Instrucción while()

**Ejemplo 01:** Listar un número desde `1` hasta `varnumero`

```python
varnumero = int( input("Ingrese el número: ") )
i = 1
while (i <= varnumero):
	print("numero: ",i)
	i = i + 1
```

**Ejemplo 02:** Listar sólo los números pares desde `2` hasta `varnumero`

```python
varnumero = int( input("Ingrese el número: "))
i = 1
while (i <= varnumero):
	if (i % 2 == 0): # filtro para números pares
		print("número: ", i)
	i = i + 1 # indentación. No pertenece al if()
```

Calcular el área de un triángulo

<Solution>
