## Instrucción for()
En python, la instrucción `for()` permite recorrer estructuras, elemento por elemento.

**Ejemplo 1**
```python
for i in range(3):
    print(i)
```
En el ejemplo 1, la instrucción `for()` recorre la estructura `range(3)`, que generará números comanzando desde 0 hasta 2, pero no incluirá el número 3. En otras palabras, `range(3)` es `[0,1,2]`. Luego la instrucción `for()` recorre cada elemento y los imprime en líneas separadas.

**Ejemplo 2**
```python
for i in range(1,3):
	print(i)
```
En el ejemplo 2, la instrucción `for()` recorre la estructura acotada `range(1,3)`. Esta estructura acotada sigue el principio *"inicio incluyente, final excluyente"*. Esto es, generará números comenzando desde 1 hasta 2, pero no incluirá el número 3. Entonces el rango generado será `[1,2]`

**Ejemplo 3**
```python
cadena = "LuisYanez"
print("largo string: ", len(cadena))
for i in cadena:
	print(i)
```

En el ejemplo 3, la instrucción `for()` recorre la cadena de caracteres `cadena`, imprimiendo cada letra en el recorrido. También se utiliza la función `len()`, que permite contar la candidad de elementos (letras) dentro de la cadena de caracteres

# multiplicar 'digito invertida' por factor 2,...,7,2,...,7 y sumarlas

while (digito > 0):
    print(f'{digito%10} * {factor} = {digito%10 * factor}')
    suma += (digito % 10) * factor
    digito = int(digito / 10)
    factor = 2 if (factor == 7) else factor + 1
print()
print('SUMA:',suma)

# división por 11
print()
print(f'DIVISION: {suma} / 11 = {suma/11}')

# tomar solo decimales
digito = (suma/11) - int(suma/11)

# despejar 'digito'
print()
print(f'Dígito: ( 11 - ( 11 * ({digito}) ) )')
