## Ingresar datos al diccionario "estudiantes"
Para agregar nombre de estudiante y nota se ven 3 formas en clases.

**Ejemplo 01:** Utilizando función `setdefault()`
```python
estudiantes = {}
nombre = input('Ingrese el nombre: ')
nota = int(input('Ingrese la nota: '))

estudiantes.setdefault(nombre,nota)
```

**Ejemplo 02:** Utilizando función `update()`
```python
estudiantes = {}
nombre = input('Ingrese el nombre: ')
nota = int(input('Ingrese la nota: '))

estudiante1 = {nombre:nota}
estudiantes.update(estudiante1)
```

**Ejemplo 03:** Utilizando sintaxis de corchetes
```python
estudiantes = {}
nombre = input('Ingrese el nombre: ')
nota = int(input('Ingrese la nota: '))

estudiantes[nombre] = nota
```

## Recorrer el diccionario "estudiantes"
La función `for()` sirve para recorrer estructuras, por lo que se puede utilizar para recorrer un diccionario. 

- Es necesario 2 variables auxiliares `n`, `c` para capturar la `key` y `value` en el recorrido.
- Se utiliza la función `items()` para convertir el diccionario a una lista de tuplas, y así poder capturar sus datos en el recorrido.

```python
for n,c in estudiantes.items():
	print('Nombre:',n,'	Nota:',c)
```
