## Ejercicio 08
Intercalar las tuplas:

- `(1,3,5)`
- `(2,4,6)`

<details>
    <summary>Solución</summary>

```python
t1 = (1,3,5)
t2 = (2,4,6)

intercalados = [] 
for i in zip(t1,t2):
	for j in i:
		intercalados.append(j)

t3 = tuple(intercalados)
print(f't3: {t3}')
```

- En el 1er `for()`, `zip()` convierte las 2 tuplas en pares de tuplas `((1,2),(3,4),(5,6))`
- En el segundo `for()` recorre cada par de tupla y almacena sus elementos en la lista `intercalados`
- luego la lista la convierte a tupla y la almacena en `t3`

</details>

## Ejercicio 06

Sean los diccionarios:

- dic1 = \{'nombre':'Jose','direccion':'miDireccion123'\}
- dic2 = \{'ciudad':'Concepcion', 'telefono':1234567\}

Combinar los diccionarios

<details>
    <summary>Solución</summary>

```python

dic2 = {'ciudad':'Concepcion', 'telefono':1234567}

dic1.update(dic2)
print(f'dic1: {dic1}')
```
</details>

## Ejercicio 07
Recorrer con un `for()` el diccionario resultante del Ej06

<details>
    <summary>Solución</summary>

```python
dic = {
	'nombre':'Jose',
	'direccion':'miDireccion123',
	'ciudad':'Concepcion',
	'telefono':1234567
}

for i,j in dic.items():
	print(f'clave: {i}\t\t\tvalor: {j}')
```
