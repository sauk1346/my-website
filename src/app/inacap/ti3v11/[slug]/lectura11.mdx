# Actividad de Práctica

1. **¿Cuál de las siguientes situaciones representa un riesgo de seguridad en el almacenamiento de datos en Python?**
    1. Utilizar una contraseña segura para acceder a una base de datos.
    2. <span style={{color:'red'}}>Permitir la ejecución de consultas SQL dinámicas sin parámetros.</span>
    3. Limitar el acceso a los datos solo a usuarios autorizados.
    4. Mantener copias de seguridad regulares de los datos.

2. **¿Cuál de los siguientes es un riesgo de seguridad asociado con la validación de datos en Python?**
    1. <span style={{color:'red'}}>No realizar la validación de entrada de usuario.</span>
    2. Asegurar que los datos ingresados cumplan con el formato esperado.
    3. Utilizar funciones de hash para proteger contraseñas.
    4. Utilizar cifrado SSL/TLS para proteger la transferencia de datos.

3. **¿Cuál de las siguientes técnicas ayuda a mitigar el riesgo de divulgación de información sensible en Python?**
    1. Utilizar encriptación para proteger los datos en tránsito y en reposo.
    2. Realizar auditorías de seguridad periódicas para identificar vulnerabilidades.
    3. <span style={{color:'red'}}>Implementar controles de acceso para restringir quién puede acceder a los datos.</span>
    4. Proporcionar mensajes de error claros y seguros para ayudar a los usuarios.

4. **¿Qué medida de seguridad ayuda a mitigar el riesgo de Cross-Site Scripting (XSS) en Python?**
    1. <span style={{color:'red'}}>Escapar o codificar correctamente los datos antes de mostrarlos en una página web.</span>
    2. Utilizar validación de entrada del usuario para evitar la introducción de scripts maliciosos.
    3. Utilizar funciones de hash para proteger los datos confidenciales.
    4. Implementar controles de acceso para restringir el acceso a los datos.

5. **¿Cuál de las siguientes acciones ayuda a mitigar el riesgo de desbordamiento de búfer en Python?**
    1. Utilizar encriptación para proteger los datos en tránsito.
    2. Realizar copias de seguridad regulares de los datos.
    3. <span style={{color:'red'}}>Validar la entrada del usuario para asegurarse de que no exceda ciertas longitudes.</span>
    4. Implementar funciones de hash para proteger las contraseñas.
