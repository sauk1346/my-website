## Términos

1. **Personal Computer (PC):** Computador estándar de escritorio.
2. **All-in-One (AIO):** Computador de escritorio, integrado en un monitor.
3. **Hardware:** Parte física de un computador o dispositivo electrónico.
4. **Software:** Conjunto de programas, instrucciones que operan los dispositivos electrónicos y permiten la ejecución de tareas específicas.
5. **Firmware:** Es un tipo de software que está permanentemente almacenado en un dispositivo de hardware. Se encuentra en la memoria de "Sólo Lectura" (ROM), memoria flash u otro tipo de memoria no volátil.
6. **Device Driver:** Controlador de dispositivo. Permite al OS reconocer, controlar y utilizar un componente de hardware.
7. **Disco Duro SATA (Serial Advanced Technology Attachment):** SATA es la interfaz de transferencia de datos. Velocidad de lectura 600 Mb/s. Utilizan discos giratorios y cabezales de lectura/escritura mecánicos.
8. **Disco Duro SSD (Solid State Drive):** Utiliza memoria flash para guardar datos de forma persistente.
9. **Disco Duro SSD M2:** Tecnología M2 permite que sean más compactos y se conecten directamente a la placa madre sin necesidad de cables adicionales.
10. **Fuentes de poder:** Suministra alimentación a todo el sistema.
11. **Placa madre:** Circuito principal dentro de una PC. Contiene la mayoría de la electrónica. Permite conectar y coordinar todos los componentes.
12. **Memoria RAM (Random Access Memory):** Memoria para almacenar temporalmente datos e instrucciones que la CPU necesita acceder rápidamente durante la ejecución de programas y tareas.
13. **Adaptador WIFI (Wireless Fidelity):** Permite a una computadora o dispositivo electrónico conectarse a una red inalámbrica.
14. **Teclado, Mouse:** Dispositivos de entrada. Permiten la interacción del usuario con el sistema.
15. **VGA (Video Graphics Array):** Estándar de interfaz de video. Su característica principal son 3 filas de agujeros, 15 en total.
16. **DVI (Digital Visual Interface):** Estándar de interfaz de video. Puede transmitir señales de video tanto digitales como analógicas.
17. **Display Port:** Estándar de interfaz digital. Permite la transmisión de audio y video a través de un mismo cable. Es común en monitores de computador y dispositivos informáticos.
18. **HDMI (High Definition Multimedia Interface):** Estándar de interfaz digital. Es común en smartTVs, consolas de videojuegos, dispositivos de streaming.
19. **USB (Universal Serial Bus):** Puerto multipropósito. Se utiliza en una variedad de dispositivos. Existen 3 modelos: tipo-A, tipo-B, tipo-C. Los tipo-C son de última generación.
