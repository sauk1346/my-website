# Sistemas Operativos

## Windows
Es una familia de sistemas operativos, desarrollada por Microsoft. Ampliamente utilizado en todo el mundo debido a su compatibilidad, versatilidad y familiaridad para los usuarios. 

### cmd
*Command Prompt*, tambien conocido como *cmd* o *símbolo del sistema*. Es una interfaz de línea de comandos (CLI).

Permite realizar tareas complejas con comandos simples, ahorrando tiempo y esfuerzo en comparación con la interfaz gráfica (GUI).

### comandos básicos
- **ipconfig:** Muestra la configuración de todas las interfaces de red activas en el sistema
- **ping \<dirección_ip o nombre_de_dominio\>:** Verifica la conectividad entre 2 dispositivos a través de una red IP.
- **ping 127.0.0.1:** *Loopback*. un loopback es la capacidad de un PC para enviarse datos a sí mismo a través de la red. Entonces, esta instrucción sirve para realizar pruebas de conectividad en un equipo.

### Administrador de Tareas
*Task Manager*. Permite a los usuarios ver y gestionar los procesos en ejecución. Es una herramienta escencial para monitorear y controlar el comportamiento del sistema, así como para solucionar problemas.

## Linux

Es un sistema operativo de código abierto basado en UNIX. Se utiliza en una amplia variedad de dispositivos, desde servidores y supercomputadoras hasta teléfonos móviles y dispositivos integrados.

Algunas de sus distribuciones más conocidas son:

- Ubuntu
- Fedora
- Debian
- CentOS
- Red Hat
