# Higiene Digital, Tipos de Licencia, Versiones de Windows 10

## Higiene digital
Se refiere a la práctica y hábitos necesarios para mantener dispositivos y datos seguros. Algunos conceptos y herramientas utilizadas para este propósito son:

- **Rollback:** Revertir un sistema, base de datos o software a un estado previo debido a un error. Es una medida de recuperación para asegurar que el sistema vuelva a un estado estable y funcional.
- **Windows Defender:** Software integrado en Windows. Protege contra una variedad de amenazas, como virus, malware, spyware y otras formas de software malicioso.
- **Firewall:** Herramienta de seguridad de red. Supervisa y controla el tráfico de red entrante y saliente. Protege las redes y dispositivos frente a accesos no autorizados.
- **12Ghosts Wash (<ExternalLink href="https://12ghosts-wash.software.informer.com/">https://12ghosts-wash.software.informer.com/</ExternalLink>):** Es una herramienta de limpieza y mantenimiento para sistemas Windows. Permite eliminar archivos temporales, limpiar historial y caché, borrado seguro, entre otros.
- **dxdiag:** instrucción en CMD. Herramienta de diagnóstico. Proporciona información detallada sobre los componentes y controladores DirectX instalados.


## Tipos de licencia
1. **De software propietario:** El usuario paga para usarlo. No tiene acceso al código fuente.
2. **De código abierto:** El software y código son de acceso libre. Usuarios pueden modificarlo y distribuirlo.
3. **De software gratuito:** Sin costo pero con limitaciones. Puede incluir publicidad invasiva.
4. **De software freeware:** Como el 'gratuito', pero con menos restricciones.
5. **De software comercial:** Se compra para obtener funciones avanzadas y soporte. El código fuente puede o no ser de acceso restringido, por lo que una licencia de 'software comercial' también puede ser de 'software propietario'.

## Versiones de Windows 10

|Version|Descripción|
|---|---|
|Windows 10 Home|Sólo admite hasta 128GB de RAM|
|Windows 10 Pro|Puede reconocer hasta 2TB de RAM, entorno 'windows sandbox'|
|Windows 10 Enterprise|Una mejora para la versión 'Pro'. Sistema de archivos 'ResFS'|
|Windows LTSC: *Canal de servicio a largo plazo*|Recibe actualizaciones cada 2 años. Pensada para evitar cualquier falla. Útil para evitar modificaciones o incompatibildad que puedan surgir|
|Windows 10 Education|Pensada para instituciones educativas|
|Windows 10 Workstations|Para altos recursos. Soporta hasta 6TB de RAM|
|Windows 10 ARM|Modificación de la versión 'Pro' para funcionar con procesadores ARM|
|Windows 10 IoT|Pensada para dispositivos embebidos (cajeros automáticos, máquinas expendedoras, ascensores, dispositivos de red, etc.)|
|Windows 10 Team|Pensada para las pizarras digitales 'Surface Hub'|
|Windows 10 X|Pensada para dispositivos de pantalla doble|
|Windows 10 S|Oficialmente ya no existe, es un modo. Sólo permite instalar aplicaciones de la tienda Microsoft|
