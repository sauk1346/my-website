# Máquina Virtual: Pasos

1. Descargar [VirtualBox](https://www.virtualbox.org/) 
2. Descargar ISO de Windows 10
3. Obtener número-serie de Windows 10
4. Crear máquina virtual utilizando VirtualBox (próx clase)

## Requerimientos

- PC de al menos 8GB RAM.
- Espacio suficiente en disco duro.
- Procesador con soporte de virtualización (Intel-VT o AMD-V).
