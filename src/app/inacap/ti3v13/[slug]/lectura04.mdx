# Actividad de Práctica 04

1. **¿Cuál es la finalidad del mantenimiento preventivo en computadoras personales, portátiles y dispositivos móviles?**
    1. Resolver problemas existentes.
    2. Documentar los fallos de hardware.
    3. <span style={{color:'red'}}>Evitar posibles fallas y prolongar la vida útil.</span>
    4. Desconectar componentes innecesarios.

2. **¿Cuál de las siguientes es una mala práctica de mantenimiento preventivo en computadoras personales, portátiles y dispositivos móviles?**
    1. Actualizar regularmente el sistema operativo y las aplicaciones.
    2. <span style={{color:'red'}}>Ignorar alertas de seguridad y no instalar parches de software.</span>
    3. Realizar copias de seguridad periódicas de los datos importantes.
    4. Limpiar físicamente los componentes internos de forma inadecuada.

3. **¿Cuál de las siguientes acciones ejemplifica una buena práctica al utilizar una guía de documentación?**
    1. No documentar en la configuración del sistema para ahorrar tiempo.
    2. <span style={{color:'red'}}>Registrar regularmente las fechas de vencimiento de la licencias de software.</span>
    3. Mantener una única copia de seguridad de los registros en un dispositivo externo.
    4. No revisar los registro s de mantenimiento a menos que ocurra un problema.
