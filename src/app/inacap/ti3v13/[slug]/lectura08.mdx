# Actividad de Práctica 08

1. **¿Cuál es el propósito principal de realizar particiones en una unidad de almacenamiento?**
    1. Mejorar la velocidad de la computadora.
    2. <span style={{color:'red'}}>Organizar y separar datos de manera eficiente.</span>
    3. Aumentar la capacidad de almacenamiento.
    4. Desfragmentar archivos.

2. **¿Qué proceso ocurre cuando se formatea la unidad principal de disco duro?**
    1. Se realiza la separación lógica de datos.
    2. Se organiza mejor el sistema operativo y aplicaciones.
    3. <span style={{color:'red'}}>Se eliminan todos los datos existente.</span>
    4. Se elimina el sistema operativo.

3. **¿Cuál sistema de archivos comúnmente aplica a particiones primarias en el entorno Windows?**
    1. FAT32
    2. <span style={{color:'red'}}>NTFS</span>
    3. Ext4
    4. XFS

4. **¿Cuál es el requisito fundamental para la partición de arranque de un sistema operativo?**
    1. <span style={{color:'red'}}>Configurarse a una partición primaria.</span>
    2. Configurarse a una partición extendida.
    3. Designarse a partición lógica.
    4. Poseer tamaño mínimo de 100MB.

5. **¿Cuál es el comando para formatear una partición de disco duro en el sistema operativo Windows?**
    1. diskpart
    2. <span style={{color:'red'}}>format</span>
    3. fdisk
    4. mkfs
