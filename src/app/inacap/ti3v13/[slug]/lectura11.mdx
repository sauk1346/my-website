# Actividad de Práctica 11

1. **¿Por qué es importante realizar copias de seguridad de datos en un sistema operativo?**
    1. Ocupar más espacio de almacenamiento.
    2. <span style={{color:'red'}}>Prevenir la pérdida de datos en caso de fallo del sistema.</span>
    3. Acelerar el rendimiento del sistema.
    4. Mejorar la seguridad contra virus.

2. **¿Cuál es una buena práctica al generar copias de seguridad de un sistema operativo?**
    1. Realizar copias solo una vez al año.
    2. Guardar todas las copias en el mismo dispositivo.
    3. <span style={{color:'red'}}>Verificar la integridad de las copias regularmente.</span>
    4. Ignorar la copia de archivos del sistema.

3. **¿Qué tipo de copia de seguridad solo respalda los datos que han cambiado desde la última copia diferencial?**
    1. Completa
    2. Diferencial
    3. <span style={{color:'red'}}>Incremental</span>
    4. Continua

4. **Se requiere generar una copia de seguridad de datos en Win11, ¿Cuál es el procedimiento correcto a ejecutar?**
    1. Centro de Control > Sistema > Copias de seguridad
    2. Panel de Control > Sistema y seguridad > Seguridad y mantención
    3. Centro de Control > Sistema > Imagen del sistema
    4. <span style={{color:'red'}}>Panel de Control > Sistema y seguridad > Copias de seguridad</span>

5. **¿Cuál de las siguientes es una mala práctica al generar copias de seguridad?**
    1. Realizar una programación regular de copias de seguridad.
    2. Uso de contraseñas fuertes o cifras las copias de seguridad.
    3. <span style={{color:'red'}}>Almacenar todas las copias de seguridad en el mismo lugar que los datos originales.</span>
    4. Incluir todos los datos críticos o información importante.
