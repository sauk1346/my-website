# Actividad de Práctica 16

1. **¿Cuál de las siguientes afirmaciones es cierta acerca de la seguridad en una red inalámbrica?**
    1. Desactivar la encriptación mejora la seguridad
    2. El SSID no es relevante para la seguridad
    3. <span style={{color:'red'}}>WPA2 es un protocolo de seguridad inalámbrica</span>
    4. Un firewall no es necesario en redes inalámbricas

2. **¿Cuál es el propósito principal de una dirección MAC en una red?**
    1. Identificar un dispositivo a nivel mundial
    2. Asignar direcciones IP automáticamente
    3. Determinar la ubicación geográfica de un dispositivo
    4. <span style={{color:'red'}}>Identificar de manera única un dispositivo en la red</span>

3. **¿Cuál es una medida efectiva para configurar la seguridad de un router inalámbrico?**
    1. <span style={{color:'red'}}>Configurar WPA3 o WPA2 con una clave fuerte</span>
    2. Desactivar completamente la seguridad para facilitar el acceso
    3. Utilizar un cifrado WEP para proteger la red
    4. Permitir acceso sin contraseña para usuarios conocidos

4. **¿Por qué es importante cambiar la contraseña predeterminada de un router inalámbrico?**
    1. La contraseña predeterminada es segura y no requiere cambios
    2. Facilita el acceso a la red para todos los usuarios
    3. <span style={{color:'red'}}>La contraseña predeterminada es conocida públicamente</span>
    4. Cambiar la contraseña no tiene impacto en la seguridad de la red

5. **¿Qué medida de seguridad inalámbrica es recomendada para proteger una red Wi-Fi contra accesos no autorizados?**
    1. WEP (Wired Equivalent Privacy)
    2. <span style={{color:'red'}}>MAC Filtering (Filtrado de direcciones MAC)</span>
    3. Desactivar completamente la contraseña de la red
    4. Transmitir el nombre de la red (SSID) de forma pública
