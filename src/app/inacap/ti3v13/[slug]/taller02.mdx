# Taller Instalación Sistema Operativo W10

1. **Modificar booteo de computador de tal forma que comience desde el USB.**
	1. Insertar USB antes de Prender PC.
	2. Prender PC y presionar constantemente F12 para entrar a la BIOS.
	3. Una vez en la BIOS, seleccionar la unidad de USB.

2. **Al iniciar el instalador de Windows 10, elimine todas las particiones existentes.**
	1. Paso intermedio al instalar WIN10. Borrar todas las particiones existentes.

3. **Crear una nueva partición "Principal" de 50GB.**
	1. `1GB = 1024MB`. Por 'regla de tres simples': `50GB = 51200MB`.

4. **Dejar el espacio restante de disco para generar una partición Secundaria con posterioridad.**
	1. Se deja intacta la partición restante.

5. **Al instalar Windows 10, crear usuario `INACAP`, con password: `inacap2024`.**
	1. Se rellenan datos. usuario: `INACAP`, pass: `inacap2024`.

6. **Configurar fecha y hora para que queden con los datos actuales.**
	1. Se configura fecha y hora.

7. **En el escritorio de Windows 10, haga las configuraciones para que aparezca el ícono "Este Equipo", Red, Panel de Control, Papelera de Reciclaje.**
	1. Escritorio (click-der) > Personalizar > Temas > Configuración de iconos de escritorio

8. **Haga una captura de las caracterísiticas del computador con la herramienta "Recortes" y nómbrela como "Configuración" (archivo debe ser formato jpg).**
	1. 1ra forma: cmd > `dxdiag` > `win + shift + s` y capturar imagen.
	2. 2da forma: Inicio (click-der) > Sistema > `win + shift + s` y capturar imagen.

9. **Active el resto del espacio de disco duro disponible (que quedó pendiente en el pto 4.), formatéelo y nómbrelo como `Datos User`.**
	1. Inicio (click-der) > Administración de discos > Nuevo volumen simple. Etiqueta de volumen: `Datos User`.

10. **Cree un usuario `alumno` y asígnele permisos de "invitado".**
	1. Inicio (click-der) > Configuración > Cuentas > Familia y otros usuarios > Agregar a otra persona a este equipo.

11. **¿Qué versión de Windows 10 se instaló en este computador?**
	1. Inicio (click-der) > Sistema > Especificaciones de Windows.
