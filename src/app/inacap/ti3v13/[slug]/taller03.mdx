# Armado de Red  Inalámbrica

## Cisco Packet Tracer
Es una herramienta de simulación de redes. Permite crear, configurar y simular redes informáticas de manera virtual, sin la necesidad de hardware físico real.

## Conexión inalámbrica

1. Agregar 3 laptops 
    <TerminalOutput>
        {`[End devices] > Laptop-PT`}
    </TerminalOutput>

2. Agregar 1 router
    <TerminalOutput>
        {`[Network devices] > [Wireless devices] > HomeRouter-PT-AC`}
    </TerminalOutput>

3. Establecer conexión entre dispositivos
    <TerminalOutput>
        {`[Connections] > Automatically Choose Connection Type`}
    </TerminalOutput>

6. Verificar conexión entre dispositivos mediante instrucción `ping`

    <TerminalOutput>
        {`laptop > desktop > Command Prompt`}
    </TerminalOutput>
    
```sh
ping <device_ip_address>
```
