## Ejercicio 01
Desarrolle un programa que permita ingresar 2 números, y despliegue si son amigos

- **Números amigos:** dos números son considerados números amigos si cumplen que, la suma de los divisores propios de uno de los números es igual al otro número, y viceversa
	- Ejemplo: los números 220 y 284 son una pareja de números amigos
	- Los divisores propios de 220 (excluyendo 220) son: 
		- 1, 2, 4, 5, 10, 20, 22, 44, 55, 110. Al sumarlos da 284
	- Los divisores propios de 284 (excluyendo 284) son: 
		- 1, 2, 4, 71, 142. Al sumarlos da 220
	- La suma de los divisores de 220 da 284, y la suma de los divisores de 284 da 220. Por lo tanto son números amigos

<details>
	<summary>Solución</summary>

```python
num1 = int(input("Ingrese el primer número: "))
num2 = int(input("Ingrese el segundo número: "))

suma_div_num1 = 0
for i in range(1, num1):
	if num1 % i == 0:
		suma_div_num1 = suma_div_num1 + i 

suma_div_num2 = 0
for i in range(1, num2):
	if num2 % i == 0:
		suma_div_num2 = suma_div_num2 + i

if suma_div_num1 == num2:
	print(num1," es amigo de ", num2)
else:
	print(num1," NO es amigo de ", num2)

if suma_div_num2 == num1:
	print(num2," es amigo de ", num1)
else:
	print(num2," NO es amigo de ", num1)
```

## Ejercicio 02
Desarrolle un programa que permita ingresar 10 números. Despliegue el mayor de ellos. No utilizar listas

<details>
	<summary>Solución</summary>

```python
num_max = int(input("Ingrese el núero [1 de 10]: "))

for i in range(2, 10+1):
    num = int(input(f"Ingrese el número [{i} de 10]: "))
    if num > num_max:
        num_max = num

print(f"Número Mayor: {num_max}")
```
