# Listas

## Crear una lista
```python
numeros = [4,7,9,2,3]
print(numeros)
```

## Crear una lista vacía
```python
numeros = []
print(numeros)
```

## Agregar elemento a la cola

`.append(x)`: agrega el elemento-x al final de la lista

```python
numeros = [4,7,9,2,3]
numeros.append(1)
print(numeros)
```

## Agregar elemento en posición específica

`.insert(x,y)`: agrega en el índice-x, el elemento-y

```python
numeros = [4,7,9,2,3]
numeros.insert(2,"cinco")
print(numeros)
```

## Eliminar elemento(s)

1. `del numeros[x]` Elimina el elemento en el índice-x, de la lista numeros

```python
numeros = [4,7,8,2,3]
del numeros[2]
print(numeros)
```

2. `del numeros[:]` elimina todos los elementos de la lista numeros

```python
numeros = [4,7,8,2,3]
del numeros[:]
print(numeros)
```

## Contar elementos
`len(x)` cuenta todos los elementos de la lista-x

```python
numeros = [4,7,8,2,3]
print(f"Total elementos: {len(numeros)}")
```

## Referenciar lista
`lista2 = numeros`

- Puede parecer que `lista2` es una copia de `numeros`, pero es una referencia
- Una referencia significa que, los cambios que se hagan en ella, modifican a la original

```python
numeros = [4,7,8,2,3]
lista2 = numeros

del lista2[:]
print(numeros)
```

## Copiar lista

`copia = numeros[:]`

- Los cambios que se hagan en la lista `copia`, no afectan a la lista `numeros`

```python
numeros = [4,7,8,2,3]
copia_lista = numeros[:]

del copia_lista[:]
print(numeros)
```

## Copiar un rango de elementos

`copia_rango = numeros[i,j]`

- Copia elementos desde el índice-i (inclusivo) hasta el índice-j (exclusivo)

```python
numeros = [4,7,8,2,3]
copia_rango = numeros[1:3]

print(copia_rango)
```

## Recorrer lista
Se realiza con ayuda de un ciclo `for`

```python
numeros = [4,7,9,2,3]
for elem in numeros:
	print("Elemento ->", elem)
```

## Calcular Índices
Se utiliza el ciclo `for` y el método `len()`

```python
numeros = [4,7,8,2,3]
for indice in range(len(numeros)):
	print("Elemento en índice", indice, "->", numeros[indice])
```

## Último elemento

1. `numeros[len(numeros)-1]`
2. `numeros[-1]`

Ambas instrucciones entregan el último elemento de una lista

```python
numeros = [4,7,8,2,3]

ultimo_ele = numeros[len(numeros)-1]
ult_elem = numeros[-1]

print(ultimo_ele)
print(ult_elem)
```

# Ejercicios

## Ejercicio 01
Crear una lista con 10 números aleatorios del 0 al 9. Desplegar solo los números mayores al último número

<details>
	<summary style={{color:'red'}}>Solución</summary>

```python
import random

numeros = [random.randint(0,9) for e in range(10)]
print(f"\nNúmeros aleatorios: {numeros}")

ultimo = numeros[-1]
print(f"Números mayores que {ultimo}: ", end="")
for e in numeros:
    if e > ultimo:
        print(str(e) +" ", end="")
```
