### MENÚ DE LOGIN

1. Iniciar la aplicación con la siguiente instrucción

```sh
python main.py
```

2. En el Menú desplegado (Login), elegir la opción deseada

3. Ingresar nombre de usuario y contraseña. Éstos se encuentran como datos de prueba en la base de datos `buses_rios_del_sur.db`
