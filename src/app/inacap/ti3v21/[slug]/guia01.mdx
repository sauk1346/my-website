### Ejercicio 02
Supongamos que el IPC de los meses de Febrero y Marzo fueron 0.3% y 0.6% respectivamente. Crear un algoritmo en diagrama de flujo que muestre el valor de un producto actualizado y la diferencia de precio entre el mes de Febrero y Marzo

<Solution>
El valor actualizado $(V)$ de un producto $(P)$ en función del IPC $(IPC\%)$ es:

$$
\begin{aligned}
    V = P \cdot \left(1 + \frac{IPC\%}{100}\right)
\end{aligned}
$$

```python
def validar_precio():
    while True:
        try:
            precio = float(input("Ingrese el valor del producto: "))
        except ValueError:
            print("Error. Valor no válido")
        else:
            if (precio <= 0):
                print("Error. Valor no válido")
            else:
                return precio

def actualizar_precio(precio, ipc):
    return ( precio * (1 + ipc/100) )

def diferencia_precio (precio1, precio2):
    return (abs(precio1 - precio2))

ipc_feb = 0.3
ipc_mar = 0.6

precio = validar_precio()
precio_feb = actualizar_precio(precio,ipc_feb)
precio_mar = actualizar_precio(precio, ipc_mar)
diferencia = diferencia_precio(precio_feb,precio_mar)

print(f"\nValor producto actualizado a Febrero: {precio_feb:.2f}")
print(f"Valor producto actualizado a Marzo: {precio_mar:.2f}")
print(f"Diferencia precio Febrero-Marzo: {diferencia:.2f}")
```
</Solution>


### Ejercicio 03
Confeccione un algoritmo en diagrama de flujo que al leer el neto de una factura, calcule el IVA y entregue como salida el total de la factura.

<Solution>
El calculo del IVA de un producto base $(P)$, según una tasa $(t)$ es:

$$
\begin{aligned} 
    IVA = P \cdot t
\end{aligned}
$$

Donde t = 0.19 en Chile

```python

import random
import string

def generarCalcomaniasMexico(cantidad):
    calcomanias = []
    for _ in range(cantidad):
        # Formato típico: 3 letras - 3 dígitos
        letras = "".join(random.choices(string.ascii_uppercase, k=3))
        numeros = "".join(random.choices(string.digits, k=3))
        calcomania = f"{letras}-{numeros}"
        calcomanias.append(calcomania)
    return calcomanias

def calcularColor(calcomanias):

    rosa = 0
    roja = 0
```
