## Agregación y Composición

- **Agregación**
	- *objeto1* contiene o agrupa con *objeto2*, pero ambos pueden existir de forma independiente.
	- Si *objeto1* se destruye, *objeto2* puede seguir existiendo.
	- **Ejemplo:** El contenido se puede impartir en una asignatura. Si la asignatura deja de existir, el contenido aún existe (reutilizable).

<img 
    src="/ti3v21/images/fig21.png"
    style={{ display: 'block', margin: '0 auto', width:'70%' }} 
/>
<br/>
- **Composición**
	- *objeto1* depende de *objeto2* de forma inseparable
	- Si *objeto1* se destruye, *objeto2* también
	- **Ejemplo:** La sección permite organizar la asignatura. Si la asignatura deja de existir, también lo hace la sección (no tienen propósito fuera de ella).
