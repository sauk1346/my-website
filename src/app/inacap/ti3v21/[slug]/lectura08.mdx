# Manipulación de registros
Se utilizará la librería PyMySql para la manipulación de registros de datos en bases de datos. Esto mediante operaciones CRUD (crear, leer, actualizar, eliminar)

## Métodos clave de la librería PyMySql

|Nombre Método|Descripción|
|-|---|
|`execute()`|Comando de consulta para ejecutar una base de datos|
|`fetchone()`|recupera una sola fila del resultado de una consulta|
|`fetchmany(size)`|recupera una cantidad `size` de filas del resultado de una consulta|
|`fetchall()`|recupera todas las filas del resultado de una consultas|
|`rowcount`|obtiene el número de filas de la última operación|
|`close()`|cierra la conexión|


