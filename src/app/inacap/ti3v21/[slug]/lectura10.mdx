# Scripts Seguros en Python

## Librería bcrypt
- Diseñada para trabajar con funciones hash seguras
- Proporciona una forma segura de almacenar y verificar contraseñas en aplicaciones

### Instalación

```python
pip install bcrypt
```

### Funcionamiento
1. **Salt:** cuando un usuario crea una nueva contraseña, `bcrypt` genera una cadena aleatoria llamada "Salt", la cual se combina con la contraseña del usuario antes de aplicar la función hash. Esto permite que, incluso si 2 usuarios tienen la misma contraseña, sus hashes serán diferentes debido a las sales únicas.
2. **Cost Factor:** `bcrypt` utiliza un parámetro llamado "cost factor"/"work factor" para encontrar la cantidad de tiempo tarda en generar el hash. Cuanto mayor sea el cost factor. más lento será el proceso de hash. Esto se hace para aumentar la resistencia contra ataques de fuerza bruta o de diccionario.
3. **Generación del hash:** La contraseña del usuario y la sal se combinan. Luego se pasa a la función de hash interna de `bcrypt`, utilizando el cifrado Blowfish
4. **Formato del hash almacenado:** El hash resultante se almacena junto con la sal y el cost factor en un formato específico.
5. **Verificación de contraseña:** Cuando un usuario intenta iniciar sesión, la sal almacenada se extrae del hash almacenado. La contraseña ingresada por el usuario se combina con esta sal y se pasa a la misma función del hash. El hash resultante se compara con el hash almacenado. Si coinciden, la contraseña es válida.
