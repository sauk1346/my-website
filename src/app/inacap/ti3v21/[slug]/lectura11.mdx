# Librerías para Consumo de Datos Externos

## ¿Por qué?
Es una práctica común en el desarrollo de software, especialmente cuando se trabaja con servicios web, bases de datos externas, APIs u otros recusros remotos.

## Librería Request

```sh
pip install requests
```

- Permite realizar solicitudes HTTP de manera sencilla y eficiente
- Ejemplos: `GET`, `POST` y otros métodos
