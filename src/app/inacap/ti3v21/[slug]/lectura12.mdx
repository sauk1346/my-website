# Clase de Python con deserialización de información

## ¿Qué es?
La deserialización nos permite tomar datos en formato JSON o XML, y convertirlos en objetos que Python puede entender y manipular fácilmente.

## json.dumps()
Toma un objeto python (diccionario, lista) y lo convierte en una cadena de texto JSON

```python
import json

datos = {"nombre": "Armando", "edad": 24, "ciudad": "La Serena"}
cadena_json = json.dumps(datos, indent=4)
print(cadena_json)
```


## json.loads()
Toma una cadena de texto JSON y lo convierte en un objeto Python

```python
import json

cadena_json = '{"nombre": "Armando", "edad": 24, "ciudad": "La Serena"}'
datos = json.loads(cadena_json)
print(datos)
```

## json.dump()
Similar a `json.dumps()`. Pero además permite guardar la conversión en un archivo

```python
import json

datos = {"nombre": "Armando", "edad": 24, "ciudad": "La Serena"}
with open("datos.json", "w") as archivo:
	json.dump(datos.archivo)
```
