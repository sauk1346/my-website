### SCRUM
- Ejemplo de uso: "Desarrollo de una aplicación móvil para la gestión de tareas y colaboración en equipos"
- Sprints cortos
- Capacidad de gestionar cambios rápidos en los requisitos

### Kanban
- Ejemplo de uso: "Mentenimiento y mejoras continuas en un sistema de comercio electrónico"
- Se ajusta al flujo constante de trabajo y facilita la gestión de prioridades en un entorno que evoluciona rápidamente

### XP (Programación Extrema)
- Ejemplo de uso: "Desarrollo de un sistema de control de versiones"
- XP es apropiado debido a sus prácticas de desarrollo centradas en la calidad y la retroalimentación constante

### Lean
- Ejemplo de uso: "Desarrollo de un software de gestión de inventario"
- Lean es beneficioso para optimizar procesos, eliminar desperdicios y mejorar la eficiencia en la entrega de funcionalidades

### Crystal
- Ejemplo de uso: "Creación de un sistema de reservas para una pequeña empresa"
- Puede adaptarse a un equipo pequeño, priorizando la comunicación y la simplicidad

## FDD (Feature-Driven Development)
- Ejemplo de uso: "Construcción de un sistema de gestión de contenido"
- Permite desarrollar características específicas de manera incremental y guiada por funcionalidades

## Metología SCRUM
Se centra en la flexibilidad y la adaptabilidad para abordar proyectos complejos de desarrollo de software y productos
