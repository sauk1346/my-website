# Metodolgía Kanban

## Kanban
- Proviene del japones y significa "tarjeta visual" o "tablero visual"
- Se aplicaba en la producción para optimizar la eficiencia en la cadena de montaje
- Hoy se ha expandido más allá de la fabricación, desde el desarrollo de software hasta la gestión de proyectos y la planificación de tareas

### Objetivo
Limitar el trabajo en curso (WIP), para evitar la congestión y mejorar la velocidad de entrega

## Pasos para implementar Kanban
1. Definir el flujo de trabajo de los proyectos
2. Visualizar las fases del ciclo de producción
3. *Stop Starting, Start Finishing*: Priorizar el trabajo que está en curso en lugar de iniciar nuevas tareas
4. Control de flujo: Mantener un flujo de trabajo constante y hacer un seguimiento pasivo del trabajo realizado

## Softwares para Kanban
- Trello
- Kanbanize
- Jira Software
- Asana
- Monday.com
- LeanKit
