# Definición de Entregables Iniciales

## Qué es
Fase crucial que implica definición, organización y programación de tareas para garantizar la consecución exitosa del proyecto

## Etapas

### Definición de Objetivos y Alcance
1. Identificar y comprender claramente los objetivos del proyecto.
2. Establecer los límites del proyecto, definir las funciones y características clave del software.

### Descomposición del Trabajo
1. Dividir el proyecto en tareas más pequeñas y manejables.
2. Utilizar la descomposición del trabajo para crear una estructura de desglose del trabajo (EDT) que organice las actividades de manera jerárquica.

### Estimación de Tiempos y Recursos
1. Estimar la cantidad de tiempo necesario para completar cada tarea.
2. Asignar recursos, incluyendo personal, herramienta y tecnologías específicas para cada tarea.

### Secuenciación de Actividades
1. Determinar la secuencia lógica en la que deben llevarse a cagbo las actividades.
2. Identificar dependencias entre tareas y definir el orden en que deben completarse.

### Creación de un Cronograma
1. Utilizar herramientas de gestión de proyectos para elaborar un cronograma visual.
2. Representar gráficamente las fechas de inicio y finalización de cada tarea.

## Estructura de Desglose de Trabajo (EDT)
- Herramienta fundamental en la gestión de proyectos
- Permite descomponer el proyecto en componentes más pequeños y manejables

### Fases del Proyecto
1. Inicio
2. Planificación
3. Ejecución
4. Monitoreo y Control
5. Cierre

### Entregables Principales
1. Requisitos del sistema
2. Diseño de la arquitectura
3. Desarrollo del código
4. Pruebas
5. Implementación
6. Mantenimiento

### Áreas Funcionales
1. Desarrollo de software
2. Infraestructura de red
3. Administración de bases de datos
4. Seguridad de la información
5. Soporte técnico
6. Capacitación

### Tareas Específicas
1. Análisis de requisitos
2. Diseño de interfaz de usuario
3. Codificación
4. Pruebas unitarias
5. Instalación de hardware
6. Configuración de servidores

### Recursos Asignados
1. Equipo de desarrollo
2. Personal de soporte técnico
3. Especialistas en seguridad
4. Administradores de bases de datos
5. Analistas de negocios
