# Etapa de Cierre

- En TI, marca el fin de un ciclo de trabajo
- Implica la entrega formal de productos o servicios
- Evaluar y documentar todo el proyecto
- Equipos TI: consolidad, presentar logros, aprendizajes, desafíos superados, valor proporcionado al cliente u organización

## Elementos Clave Etapa de Cierre
- Entrega de productos o servicios
- Documentación y archivo
- Evaluación de desempeño
- Satisfacción del cliente
- Lecciones aprendidas
- Cierre de contratos y acuerdos

## Entregables finales
- Productos o soluciones desarrolladas
- Código fuente
- Informes de pruebas y validación
- Documentación de proyecto
- Lecciones aprendidas
- Manuales de operación
