## HTML5
Es la 5ta versión del lenguaje de marcado HTML (*HyperText Markup Language*). Introducido en 2014, revolucionó el desarrollo web con nuevas características:

- **Elementos semánticos:** header, nav, main, section, article, aside, footer
- **Multimedia nativa:** video, audio, canvas
- **Formularios mejorados:** nuevos tipos de input y validación

### Estructura básica
Todo documento HTML5 debe seguir esta estructura

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Página Web</title>
</head>
<body>
    <!-- Contenido de la página -->
</body>
</html>
``
