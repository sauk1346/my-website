# Principios de diseño responsivo de programación front end

## Material Adicional

- [glosario01.pdf](/inacap/ti3v31/docs/glosario01.pdf)
- [contenidosUnidad01.pdf](/inacap/ti3v31/docs/contenidosUnidad01.pdf)


El diseño responsivo es un enfoque en el desarrollo web moderno que asegura que las aplicaciones y sitios web se adapten de manera óptima a una variedad de dispositivos y tamaños de pantalla.

Esto se logra mediante el uso de consultas de medios (*media queries*) en CSS, que permiten aplicar diferentes estilos en función de las características del dispositivo, como:

- ancho
- altura
- orientación de pantalla

Se utilizan técnicas como:

1. Consultas de medios (*media queries*)
2. Rejillas flexibles (*flexbox*)
3. Sistemas de diseño en cuadrícula (*grid layout*)

## Mejores prácticas y consideraciones de accesibilidad
Una de las mejores prácticas es adoptar un enfoque de diseño *mobile-first*, que implica diseñar primero para dispositivos móviles y luego escalar hacia arriba para pantallas más grandes.

Esto asegura que el contenido sea accesible y utilizable en dispositivos con recursos y capacidades limitadas, y mejora la experiencia general del usuario.
