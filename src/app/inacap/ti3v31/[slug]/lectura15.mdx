# Implementación de CRUD con local storage en JS

## Local Storage
Es una **API de almacenamiento web** que permite almacenar datos en el navegador de manera persistente y sin fecha de expiración. 

Los datos almacenados en LocalStorage están disponibles incluso después de cerrar el navegador. 

Gracias a Local storage, los desarrolladores pueden almacenar información importante de sus aplicaciones web para su posterior acceso y utilización. 

### Características

- Almacenamiento clave-valor.
- Tamaño máximo de aproximadamente 5MB por dominio.
- Acceso sincrónico.

### Operaciones básicas

1. **Almacenar un valor**

```jsx
localStorage.setItem('clave','valor');
```

2. **Recuperar un valor**

```jsx
localStorage.getItem('clave');
```

3. **Eliminar un valor**

```jsx
localStorage.removeItem('clave');
```

4. **Limpiar todo el almacenamiento**

```jsx
localStorage.clear();
```
