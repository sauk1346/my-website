## Qué es una Base de Datos

Es una recopilación organizada de información

## Ventajas NoSql

- Son flexibles
- No requieren esquemas fijos

## Desventajas NoSql

- Menor trazabilidad de los datos
- Menos estandarización

## MongoDB

- Software a utilizar
- **IDE**: MongoDB Compass

## JSON

- *Javascript Object Notation*
- Formato de datos ligero. Utilizado para almacenar y transportar datos
- Muy utilizado en aplicaciones web, ya que es fácil de leer tanto para humanos como para máquinas

```json
{
    "name":"José",
    "age":"30",
    "email":"jose@gmail.com",
}
```
