# Ejercicios

Cargar los archivos [productos.csv](/ti3v32/data/productos.csv), [docentes.csv](/ti3v32/data/docentes.csv) y [alumnos csv](/ti3v32/data/alumnos.csv) a la BD

## Ejercicio 01

**Insertar un nuevo estudiante**

<details>
    <summary style={{color:'red'}}>Solución</summary>

<br/>

```json
{
  "_id": "666",
  "nombre": "Max",
  "apellido": "Power",
  "edad": "20",
  "carrera": "diseño",
  "correo": "max@example.com",
  "notas": [3, 2, 5]
}
```

## Ejercicio 07
**Actualizar el correo de un estudiante**. El alumnos Juan Pérez ha cambiado su correo. Ahora es juan.perez2025@ejemplo.com

<details>
    <summary style={{color:'red'}}>Solución</summary>

<br/>
- Seleccionar colección `alumnos`
- Ir a query y agregar:

```json
{"nombre":"Juan", "apellido":"Pérez"}
```
- Presionar botón `find`
- Presionar botón `update` y agregar:

```json
{$set:{"correo":"juan.perez333@gmail.com"} }
```
- Presionar `refresh`

</details>
