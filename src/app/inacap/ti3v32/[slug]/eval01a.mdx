### Ejercicio 2 – Filtros en la pestaña "Documents" (15 puntos)
Realiza los siguientes filtros desde la vista visual de documentos (no en shell):

1. En la colección clientes, filtra todos los clientes con edad mayor a 30.

<details>
    <summary style={{color:'red'}}>Solución</summary>

<br/>
```json
{ "edad": { "$gt": 30 } }
```
</details>

<br/>
2. En la colección productos, filtra todos los productos cuyo precio sea menor a 200.

<details>
    <summary style={{color:'red'}}>Solución</summary>

<br/>
```json
{ "precio": { "$lt": 200 } }
```
</details>

### Ejercicio 3 – Actualización visual (15 puntos)
Realiza las siguientes actualizaciones desde el botón "Edit Document":

1. Aumenta el campo stock de un producto en +5 unidades.

<details>
    <summary style={{color:'red'}}>Solución</summary>

<br/>
- Seleccionar la colección "productos"
- Buscar un producto (por ejemplo la "Laptop HP")
- Hacer clic en el ícono de edición (lápiz) de ese documento
- Cambiar el valor del campo "stock" de 15 a 20
- Hacer clic en "Update"

</details>

<br/>
2. Agrega el campo estado: "activo" a un cliente existente.

<details>
    <summary style={{color:'red'}}>Solución</summary>

<br/>
- Seleccionar la colección "clientes"
- Buscar un cliente (por ejemplo "Ana López")
- Hacer clic en el ícono de edición
- Agregar el campo "estado" con valor "activo"
- Hacer clic en "Update"

</details>

<br/>
3. A todos los productos de la categoría "Accesorios", agrega el campo proveedor: "ElectroMax".

<details>
    <summary style={{color:'red'}}>Solución</summary>
