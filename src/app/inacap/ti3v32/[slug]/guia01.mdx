### Ejercicio 01
**Creación de Base de datos y Colecciones**

- Crea una base de datos llamada Biblioteca
- Dentro de la base de datos Biblioteca, crea dos colecciones: Libros y Visitantes
- Asegúrate de que la base de datos y las colecciones se hayan creado correctamente usando MongoDB Compass

<details>
    <summary style={{color:'red'}}>Solución</summary>

<br/>
1. Ir a `LocalConn`
    - botón `+`
    - `Create Database`
    - nombre: `Biblioteca`
2. Ir a BD `Biblioteca`
    - botón `+`
    - `Create collection`
    - nombre: `Libros`
    - nombre: `Visitantes`

</details>

<br/>
### Ejercicio 02
**Inserción de Documentos**

- Inserta tres documentos en la colección Libros. Cada documento debe tener los siguientes campos:
    - título: El título del libro
    - autor: El nombre del autor del libro
    - anio: El año de publicación del libro.
    - disponible: Un valor booleano que indica si el libro está disponible para préstamo

- Inserta dos documentos en la colección Visitantes. Cada documento debe tener los siguientes campos:
    - nombre: El nombre del visitante
    - email: El correo electrónico del visitante
    - fechaRegistro: La fecha que el visitante se registró en la biblioteca

<details>
    <summary style={{color:'red'}}>Solución</summary>

<br/>
1. Seleccionar colección `Libros`
2. Presionar botón `ADD DATA` > `Insert document`
3. Ingresar los siguientes documentos:

```json
[
  {
    "título": "La sombra del viento",
    "autor": "Carlos Ruiz Zafón",
    "anio": 2001,
    "disponible": true
  },
  {
    "título": "Orgullo y prejuicio",
    "autor": "Jane Austen",
    "anio": 1813,
    "disponible": false
  },
  {
    "título": "Don Quijote de la Mancha",
    "autor": "Miguel de Cervantes",
    "anio": 1605,
    "disponible": true
  }
]
```
4. Presionar botón `Insert`
5. Seleccionar colección `Visitantes`
6. Presionar botón `ADD DATA` > `Insert document`
7. Ingresar los siguientes documentos:

```json
[
  {
    "nombre": "AAA",
    "email": "aaa@example.com",
    "fechaRegistro": { "$date":"2024-04-25T10:15:00.000Z" }
  },
  {
    "nombre": "BBB",
    "email": "bbb@example.com",
    "fechaRegistro": { "$date":"2024-04-24T14:30:00.000Z" }
  }
]
```
8. Presiona botón `Insert`

</details>

### Ejercicio 03
**Actualización de Documentos**

- Actualiza un libro en la colección Libros para cambiar su estado de disponible a false
- Añade un nuevo campo a todos los documentos en la colección Visitantes llamado miembroActivo, estableciendo su valor en true

<details>
    <summary style={{color:'red'}}>Solución</summary>

<br/>
