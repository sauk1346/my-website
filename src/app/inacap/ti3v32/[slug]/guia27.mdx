# Guía de Ejercicios - Base de estudiantes

- **Base de datos:** educacionBD
- **Colección:** estudiantes
- **CSV:** [estudiantes.csv](/inacap/ti3v32/data/estudiantes.csv)

## Ejercicio 01
Buscar estudiantes mayores de 18 años que vivan en zonas urbanas (address: "U") 

<Solution>
```js
db.estudiantes.find({ 
    age:{$gt:18}, 
    address:"U" 
}) 
```
</Solution>

## Ejercicio 02
Mostrar estudiantes que tengan failures mayores o iguales a 2 y que no reciban apoyo escolar (schoolsup: "no")

<Solution>
```js
db.estudiantes.find({ 
    failures:{$gte:2}, 
    schoolsup:"no" 
}) 
```
</Solution>

## Ejercicio 03
Buscar estudiantes con studytime igual a 1 o freetime igual a 5

<Solution>
```js
db.estudiantes.find({ 
    $or: [ 
        {studytime:1}, 
        {freetime:5} 
    ] 
}) 
```
</Solution>

## Ejercicio 12
Mostrar estudiantes con promedios de notas entre G1, G2 y G3 mayores a 15

<Solution>
```js
db.estudiantes.find({
    "$expr": {
        "$gt": [
            { "$divide": [ { "$add": ["$G1", "$G2", "$G3"] }, 3 ] },
            15
        ]
    }
})
```
</Solution>


## Ejercicio 13
Listar estudiantes con absences superiores a 10 y que tengan consumo semanal de alcohol (Walc) mayor que diario (Dalc)

<Solution>
```js
db.estudiantes.find({
    "$expr": {
        "$and": [
            { "$gt": ["$absences", 10] },
            { "$gt": ["$Walc", "$Dalc"] }
        ]
    }
})
```
</Solution>

## Ejercicio 14
Buscar estudiantes sin ninguna inasistencia (absences = 0) y calificaciones finales iguales o superiores a 15

<Solution>
```js
db.estudiantes.find({
    "absences": 0,
    "G3": {"$gte": 15}
})
```
</Solution>

## Ejercicio 15
Filtrar estudiantes cuyo rendimiento empeora en cada evaluación (G1 > G2 > G3)

<Solution>
```js
db.estudiantes.find({
    "$expr": {
        "$and": [
            { "$gt": ["$G1", "$G2"] },
            { "$gt": ["$G2", "$G3"] }
        ]
    }
})
```
</Solution>

## Ejercicio 16

```js
    romantic: { $exists: true }
})
```
</Solution>

## Ejercicio 20
Listar estudiantes que vivan en zona rural (address: "R") y cuya nota final (G3) sea igual a la nota inicial (G1)

<Solution>
```js
db.estudiantes.find({
    address: "R",
    $expr: { $eq: ["$G3", "$G1"] }
})
```
</Solution>
