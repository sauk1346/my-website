# Guía de Ejercicios - Base de estudiantes

- **Base de datos:** educacionBD
- **Colección:** estudiantes
- **CSV:** <a href='/ti3v32/data/estudiantes.csv' target="_blank" rel="noopener noreferrer" className={styles.link}>estudiantes.csv</a>

## Ejercicio 01
Buscar estudiantes mayores de 18 años que vivan en zonas urbanas (address: "U") 

<details>
    <summary style={{color:'red'}}>Solución</summary>
```js
db.estudiantes.find({ 
    age:{$gt:18}, 
    address:"U" 
}) 
```
</details>

## Ejercicio 02
Mostrar estudiantes que tengan failures mayores o iguales a 2 y que no reciban apoyo escolar (schoolsup: "no")

<details>
    <summary style={{color:'red'}}>Solución</summary>
```js
db.estudiantes.find({ 
    failures:{$gte:2}, 
    schoolsup:"no" 
}) 
```
</details>

## Ejercicio 03
Buscar estudiantes con studytime igual a 1 o freetime igual a 5

<details>
    <summary style={{color:'red'}}>Solución</summary>
```js
db.estudiantes.find({ 
    $or: [ 
        {studytime:1}, 
        {freetime:5} 
    ] 
}) 
```
</details>
