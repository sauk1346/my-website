# Operaciones esenciales con MongoDB

## Motrar bases de datos existentes
```js
show dbs
```

## Consultar base de datos actual
```js
db
```

## Crear una base de datos
```js
use bd_datos
```

## Crear colecciones
```js
db.createCollection("vehiculos")
```

## Visualizar colecciones de la base de datoa actual
```js
show collections
```

## Eliminar colecciones
```js
db.vehiculos.drop()
```

## Ayuda en MongoDB
```js
db.help()
```

## Eliminar base de datos activa
```js
db.dropDatabase()
```
