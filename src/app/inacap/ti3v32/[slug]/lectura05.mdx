# Etapas del proceso de Instalación de MongoDB

- Link de instalación: <ExternalLink href='https://www.mongodb.com/try/download/community'>https://www.mongodb.com/try/download/community</ExternalLink>

## Comandos básicos

1. **Visualización de bases de datos existentes**
```js
show dbs
```

2. **Consultar la base de datos actual**
```js
db
```

3. **Crear una base de datos**
```js
use bd_datos
```

4. **Crear colecciones**
```js
db.createCollection("vehiculos")
```

5. **Visualizar colecciones de la base de datos actual**
```js
show colections
```

6. **Eliminar colecciones**
```js
db.vehiculos.drop()
```

7. **Eliminar base de datos activa**
```js
db.dropDatabase()
```

8. **Ayuda en MongoDB**
```js
db.help()
```

9. **Limpiar pantalla**
```js
cls
```
