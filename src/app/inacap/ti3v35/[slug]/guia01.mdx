1. **Si tenemos una IP clase "B" y queremos dividirla en en 8 subredes. ¿Cúal es la máscara que necesitamos?**

<Solution>
1. Como la IP es clase "B", la estructura inicial de la máscara es la siguiente:

<SmartTable
	tableLayout="fixed"
	headers={['Sistema', 'Máscara']}
	rows={[
		['Binario', '11111111.11111111.00000000.00000000'],
		['Decimal', '255.255.0.0']
	]}
/>

2. Queremos agregar 8 subredes, por lo que debemos encontrar un valor `n` (número de bits) tal que

$$
\begin{aligned}
    2^n \ge 8
\end{aligned}
$$

3. El valor encontrado es `n = 3`. Esto quiere decir que se cambian 3 bits de la máscara de 0 a 1

<SmartTable
	tableLayout="fixed"
	headers={['Sistema', 'Máscara']}
	rows={[
		['Antes (binario)', '11111111.11111111.00000000.00000000'],
		['Ahora (binario)', '11111111.11111111.11100000.00000000'],
		['Ahora (decimal)', '255.255.224.0']
	]}
/>

$\therefore$ La máscara que necesitamos para dividir una IP de clase "B" en 8 subredes, es `255.255.224.0`. También se puede responder `/19` (número de bits)
</Solution>

3. **¿Cuál es el número de hosts disponibles para cada subred**

<Solution>
1. Identificamos los bits disponibles para hosts:

    - Mascara : /19, es decir, 19 bits de los 32 son para la red y subred
    - Bits para hosts: $n = (32 - 19) = 13\,bits$ 

2. Se aplica la siguiente fórmula para **Hosts útiles**:

$$
\begin{aligned}
    Hosts\,útiles = 2^n - 2
\end{aligned}
$$

3. Se obtiene entonces:

$$
\begin{aligned}
    Host\,útiles &= 2^{13} - 2\\[10pt]
    &= 8192 - 2\\[10pt]
    &= 8190
\end{aligned}
$$

$\therefore$ Cada una de las 8 subredes tiene 8,190 hosts útiles disponibles
</Solution>
