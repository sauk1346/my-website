# Configuración básica en la instalación de un sistema operativo Windows Server

## Windows Server
Es un sistema operativo de la compañía Microsoft desarrollado especialmente para servidores. 

## Ediciones de Microsoft Windows Server

1. **Windows Server Standard:** Esta edición es adecuada para pequeñas y medianas empresas y se enfoca en la implementación de servicios de red esenciales, como Active Directory, DNS, DHCP y servicios de archivos e impresión.
2. **Windows Server Datacenter: ** Esta edición está diseñada para empresas que necesitan una gran cantidad de recursos de servidor y se centra en la virtualización de servidores y el almacenamiento de datos. Incluye características como el clúster de conmutación por error y el equilibrio de carga de red.
3. **Windows Server Essentials:** Esta edición está diseñada para pequeñas empresas con hasta 25 usuarios y 50 dispositivos y ofrece una solución todo en uno para la gestión de servidores, el correo electrónico, la seguridad y el almacenamiento de archivos.
4. **Windows Server Web:** Esta edición está diseñada para alojamiento web y se enfoca en la implementación de servicios web esenciales, como servidores web, servidores de bases de datos y servicios de correo electrónico.
5. **Windows Server Storage Server:** Esta edición se enfoca en el almacenamiento de datos y ofrece una solución de almacenamiento en red (NAS) para empresas que necesitan una gran cantidad de espacio de almacenamiento de datos.

## Configuración en la instalación

1. Configurar el nuevo hostname
2. Asignar contraseña segura
3. Asignar direccionamiento IPv4
