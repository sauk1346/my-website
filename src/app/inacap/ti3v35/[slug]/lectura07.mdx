# Incorporación de la estación de trabajo en el dominio

## Configuración de esquemas de dominio Microsoft

- **Esquemas de dominio:** Estructuras lógicas que permiten organizar y gestionar los recursos de una red de manera centralizada. Para esto, se utiliza **Active Directory (AD)**.

## Importancia de realizar configuración de esquemas de dominio Microsoft
1. **Gestión Centralizada:** Permite administrar usuarios, dispositivos y políticas de seguridad desde una ubicación central, mejorando la eficiencia y la consistencia.
2. **Seguridad Mejorada:** Facilita el control de acceso y la autenticación centralizada, protegiendo los recursos críticos.
3. **Automaticación y eficiencia:** Simplifica el mantenimiento y las actualizaciones mediante scripts y políticas de grupo, aumentando la eficiencia y reduciendo errores.
4. **Resiliencia y recuperación:** Proporciona redundancia y herramientas de recuperación rápida, asegurando la continuidad del negocio.

## Pasos para configurar un dominio Microsoft
Un dominio en el contexto de Windows Server es una colección de objetos, como usuarios y dispositivos, que comparten una base de datos común y políticas de seguridad centralizadas.

Los dominios facilitan la administración de grandes redes, permitiendo a los administradores gestionar usuarios, permisos y recursos de manera eficiente.

1. Preparación del servidor
2. Instalación de *Active Directory Domain Services* (AD DS)
3. Promoción del servidor a controlador de dominio
