# Recurso integrativo de la unidad

## Licenciamientos de software

1. **Licencia de software de pago:** El usuario paga una tarifa para poder utilizar el software. Esta tarifa puede ser una compra única o una suscripción que se renueva periódicamente.
2. **Licencias de software libre y de código abierto:** Estas licencias permiten a los usuarios acceder al código fuente del software, así como modificarlo y distribuirlo sin limitaciones.

## EULA
EULA (Acuerdo de Licencia del Usuario Final) es un contrato legalmente vinculante entre el propietario del software y el usuario final, que especifica derechos y restricciones del software.

## Tipos de cuenta Linux

- **Root:** Es la cuenta administrativa más poderosa en un sistema Linux con acceso completo a todos los comandos y archivos del sistema. Se recomienda usarla con precaución para evitar cambios no intencionales. 
- **System:** Creada para manejar procesos y servicios específicos del sistema operativo, como servidores web y bases de datos, ejecutándose en segundo plano. 
- **Normal:** Es una cuenta de usuario estándar utilizadas para actividades diarias como navegar por la web, editar documentos y ejecutar aplicaciones, con permisos limitados para proteger el sistema. 
- **Network:** Diseñada para usuarios que acceden al sistema a través de la red, asociadas a servicios como SSH, FTP o Samba, permitiendo el acceso remoto a los recursos del sistema y deben estar bien configuradas y aseguradas.
