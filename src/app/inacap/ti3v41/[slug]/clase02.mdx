# Páginas Dinámicas

## 1. Ejercicio de Repaso

Generar un formulario de contacto

1. Crear una vista `contacto` en `./miapp/views.py`

    ```python
    #./miapp/views.py
    from django.http import HttpResponse

    def index(request):
    	html = """
    		<!DOCTYPE html>
    		<html lang="en">
    		<head>
    		    <meta charset="UTF-8">
    		    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    		    <title>Document</title>
    		</head>
    		<body>
    			<h1>Bienvenido al mundo de Django</h1>
    			<p>Disfruta la aventura...</p>
    		</body>
    		</html>
    	"""
    	return HttpResponse(html)

    def contacto(request):
    	html = """
    		<!DOCTYPE html>
    		<html lang="en">
    		<head>
    		    <meta charset="UTF-8">
    		    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    		    <title>Document</title>
    		</head>
    		<body>
    			<form>
    				<label>Nombre:</label>
    				<input type="text" name="nombre" />
    				<input type="submit" value="enviar" />
    			</form>
    		</body>
    		</html>
    	"""
    	return HttpResponse(html)
    ```

2. Crear una ruta para la vista `contacto` en `./miapp/urls.py`

    ```python
    #./miapp/urls.py
    from django.urls import path
    from . import views

    urlpatterns = [
        path("index/", views.index),
        path("contacto/", views.contacto),
    ]
    ```

3. Ejecutar el servidor
    ```sh
    python manage.py runserver
    ```

4. Las vistas se ejecutarán al ingresar sus respectivas rutas
    - vista `index`    -> `http://127.0.0.1:8000/miapp/index/`
    - vista `contacto` -> `http://127.0.0.1:8000/miapp/contacto/`
