# Evaluación 02

- Tema: **Plataforma de Gestión de Cursos Online**
- [Informe](/inacap/ti3v43/docs/informe.pdf)
- [Presentación](https://sauk1346.github.io/grupo04/)

La Evaluación 02 se centra en el desarrollo de una Plataforma de Gestión de Cursos Online, un sistema que permite administrar el ciclo completo de cursos virtuales. 

Este proyecto abarca desde la creación y organización de contenidos educativos hasta la gestión de usuarios, inscripciones y seguimiento del progreso académico. 

La documentación completa del trabajo se encuentra disponible en el informe técnico, donde se detallan las decisiones de diseño, arquitectura del sistema y funcionalidades implementadas.
