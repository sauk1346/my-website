# Número Palíndromo

Dado un entero `x`, devuelve `true` si `x`es un palíndromo, y `false` en caso contrario.

## Ejemplos

**Ejemplo 1:**

<TerminalOutput>
{`Input:       x = 121
Output:      true
Explicación: 121 se lee 121 de izquierda a derecha y de derecha a izquierda.`}
</TerminalOutput>

**Ejemplo 2:**

<TerminalOutput>
{`Input:       x = -121
Output:      false
Explicación: - De izquierda a derecha se lee "-121" 
               - De derecha a izquierda se lee "121-"  
               - Por lo tanto, no es un palíndromo.`}

</TerminalOutput>

**Ejemplo 3:**

<TerminalOutput>
{`Input:       x = 10
Output:      false
Explicación: - Se lee 01 de derecha a izquierda.
               - Por lo tanto, no es un palíndromo.`}
</TerminalOutput>

## Soluciones

<Solution title="Python" icon="Python">
```python
class Solution:
    def isPalindrome(self, x: int) -> bool:
        number = str(x)
        palindrome = number[::-1]

        return number == palindrome
```
</Solution>
