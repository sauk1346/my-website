# Verificar Secuencia de Preorden en Árbol de Búsqueda Binaria 

Dado un arreglo de enteros **únicos** `preorder`, devuelve `true` *si es la secuencia correcta de recorrido en preorden de un árbol de búsqueda binaria.*

> **Input:** preorder = [5,2,6,1,3]
> **Output:** false

## Soluciones

<details>
    <summary>Python</summary>
```python
class Solution:
    def verifyPreorder(self, preorder: List[int]) -> bool:
        low = float('-inf')
        stack = []

        for p in preorder:
            if p < low:
                return False
            while stack and stack[-1] < p:
                low = stack.pop()

            stack.append(p)

        return True
```
