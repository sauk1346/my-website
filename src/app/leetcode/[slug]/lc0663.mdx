# Partición Igualitaria de Árbol

Dado la `raíz` de un árbol binario, devuelve `true` *si puedes particionar el árbol en dos árboles con sumas iguales de valores después de eliminar exactamente una arista del árbol original.*

## Soluciones

<Solution title="Python" icon="Python">
```python
class Solution:
    def checkEqualTree(self, root: Optional[TreeNode]) -> bool:
        subtree_sums = []
        
        def dfs(node):
            if not node:
                return 0
            
            left_sum = dfs(node.left)
            right_sum = dfs(node.right)
            current_sum = node.val + left_sum + right_sum
            
            # Guardar suma de todos los subárboles excepto la raíz
            if node != root:
                subtree_sums.append(current_sum)
            
            return current_sum
        
        total_sum = dfs(root)
        
        # Si es impar, imposible dividir en partes iguales
        if total_sum % 2 != 0:
            return False
```
