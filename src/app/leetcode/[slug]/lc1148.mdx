# Visitas de Artículos I

**Tabla:** `Views`

<TerminalOutput>
{`+---------------+---------+
| Column Name   | Type    |
+---------------+---------+
| article_id    | int     |
| author_id     | int     |
| viewer_id     | int     |
| view_date     | date    |
+---------------+---------+

- No hay clave primaria. La tabla puede tener filas duplicadas.
- Cada fila indica que alguien (viewer_id) vió un artículo (article_id)
    escrito por un autor (author_id) en alguna fecha (view_date).

Nota: si author_id y viewer_id son iguales indican la misma persona 
        (el autor vio su propio artículo).`}
</TerminalOutput>

Escribe una solución para encontrar todos los autores que vieron al menos uno de sus propios artículos.

Devuelve la tabla de resultados ordenada por `id` en orden ascendente.

## Ejemplo

**Input:** 
<TerminalOutput>
{`Views table:
+------------+-----------+-----------+------------+
| article_id | author_id | viewer_id | view_date  |
+------------+-----------+-----------+------------+
| 1          | 3         | 5         | 2019-08-01 |
| 1          | 3         | 6         | 2019-08-02 |
| 2          | 7         | 7         | 2019-08-01 |
| 2          | 7         | 6         | 2019-08-02 |
| 4          | 7         | 1         | 2019-07-22 |
| 3          | 4         | 4         | 2019-07-21 |
| 3          | 4         | 4         | 2019-07-21 |
+------------+-----------+-----------+------------+`}
</TerminalOutput>

**Output:** 
<TerminalOutput>
{`+------+
| id   |
+------+
| 4    |
| 7    |
+------+`}
</TerminalOutput>


## Soluciones

<Solution title="Python" icon="Python">
```python
def article_views(views: pd.DataFrame) -> pd.DataFrame:

    authors_viewed_own = views[views['author_id'] == views['viewer_id']]    
    unique_authors = sorted(authors_viewed_own['author_id'].unique())
    
    return pd.DataFrame({'id': unique_authors})
```
</Solution>
