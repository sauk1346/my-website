# Primer Número Único

Tienes una cola de enteros, necesitas recuperar el primero entero único en la cola.

Implementa la clase `FirstUnique`:

- `FirstUnique(int[] nums)` inicializa el objeto con los números en la cola.
- `int showFirstUnique()` devuelve el valor del primer entero único de la cola, o `-1` si no existe tal entero.
- `void add(int value)` inserta un valor en la cola.

## Ejemplos

**Ejemplo 1:**
<TerminalOutput>
{`Input: 
    ["FirstUnique","showFirstUnique","add","showFirstUnique","add","showFirstUnique","add","showFirstUnique"]
    [[[2,3,5]],[],[5],[],[2],[],[3],[]]

Explicación: 
    FirstUnique firstUnique = new FirstUnique([2,3,5]);
    firstUnique.showFirstUnique(); // devuelve 2

## Soluciones

<Solution title="Python" icon="Python">
```python
from collections import OrderedDict
from typing import List

class FirstUnique:

    def __init__(self, nums: List[int]):
        # Diccionario para contar frecuencias
        self.count = {}
        
        # Diccionario ordenado para mantener elementos únicos en orden
        self.unique = OrderedDict()
        
        # Procesar números iniciales
        for num in nums:
            self._add_number(num)
    
    def _add_number(self, num: int) -> None:
        # Actualizar contador
        if num in self.count:
            self.count[num] += 1
            # Si ya no es único, removerlo del diccionario de únicos
            if num in self.unique:
                del self.unique[num]
        else:
            self.count[num] = 1
            # Como es la primera vez que aparece, es único
            self.unique[num] = True

    def showFirstUnique(self) -> int:
        # El primer elemento del OrderedDict es el primer único
        if self.unique:
            return next(iter(self.unique))
        return -1

    def add(self, value: int) -> None:
        self._add_number(value)
```
