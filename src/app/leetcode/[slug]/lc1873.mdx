# Calcular Bono Especial

**Tabla:** `Employees`
```markdown
+-------------+---------+
| Column Name | Type    |
+-------------+---------+
| employee_id | int     |
| name        | varchar |
| salary      | int     |
+-------------+---------+
"employee_id" es la clave primaria para esta tabla.
Cada fila de esta tabla indica el ID del empleado, nombre del empleado y salario.
```

Escribe una solución para calcular el bono de cada empleado. El bono de un empleado es el `100%` de su salario si el ID del empleado es **un número impar** y **el nombre del empleado no comienza con el carácter `M`**. El bono de un empleado es `0` en caso contrario.

Devuelve la tabla de resultados ordenada por  `employee_ide`

## Ejemplo

**Input:**
```markdown
Tabla Employees:
+-------------+---------+--------+
| employee_id | name    | salary |
+-------------+---------+--------+
| 2           | Meir    | 3000   |
| 3           | Michael | 3800   |
| 7           | Addilyn | 7400   |
| 8           | Juan    | 6100   |
| 9           | Kannon  | 7700   |
+-------------+---------+--------+
```

**Output:**
```markdown 
+-------------+-------+
| employee_id | bonus |
+-------------+-------+
| 2           | 0     |
| 3           | 0     |
| 7           | 7400  |
| 8           | 0     |
| 9           | 7700  |
+-------------+-------+
```
**Explicación:**
> Los empleados con IDs 2 y 8 obtienen 0 de bono porque tienen un "employee_id" par.
> El empleado con ID 3 obtiene 0 de bono porque su nombre comienza con 'M'.
> El resto de los empleados obtienen un bono del 100%

## Solución

<details>
    <summary>Python</summary>
```python
def calculate_special_bonus(employees: pd.DataFrame) -> pd.DataFrame:
    
    employees['bonus'] = 0
    conditions = (employees['employee_id'] % 2 == 1) & (~employees['name'].str.startswith('M'))
    
    employees.loc[conditions, 'bonus'] = employees['salary']
    
    result = employees[['employee_id', 'bonus']].sort_values('employee_id')
    
    return result
```
