# Encontrar Números Pares de 3 Dígitos

Se te da un arreglo de enteros `digits`, donde cada elemento es un dígito. El arreglo puede contener duplicados.

Necesitas encontrar **todos** los enteros **únicos** que sigan los siguientes requisitos:

- El entero consiste en la **concatenación** de **tres** elementos de `digits` en **cualquier** orden arbitrario.
- El entero no tiene **ceros al inicio**.
- El entero es **par**.

Por ejemplo, si los `digits` dados fueran `[1,2,3]`, los enteros `132` y `312` cumplen los requisitos.

Devuelve *un arreglo **ordenado** de los enteros únicos.*

## Ejemplos

**Ejemplo 1:**
> **Input:** digits = [2, 1, 3, 0]
> **Output:** [102, 120, 130, 132, 210, 230, 302, 310, 312, 320]
> **Explicación:** Todos los enteros posibles que siguen los requisitos están en el arreglo de salida. Nota que no hay enteros **impares** o enteros con **ceros al inicio**.

**Ejemplo 2:**
> **Input:** digits = [2, 2, 8, 8, 2]
> **Output:** [222, 228, 282, 288, 822, 828, 882]
> **Explicación:** El mismo dígito se puede usar tantas veces como aparezca en digits. En este ejemplo, el dígito 8 se usa dos veces cada vez en 288, 828, y 882.

**Ejemplo 3:**
> **Input:** digits = [3, 7, 5]
> **Output:** []
> **Explicación:** No se pueden formar enteros pares usando los dígitos dados.

```python
def findEvenNumbers(digits):
    # Conjunto para almacenar números únicos
    result = set()
    n = len(digits)
    
    # Iterar sobre todas las posibles combinaciones de 3 dígitos
    for i in range(n):
        # El primer dígito no puede ser 0
        if digits[i] == 0:
            continue
            
        for j in range(n):
            # No podemos usar el mismo índice dos veces
            if j == i:
                continue
                
            for k in range(n):
                # No podemos usar el mismo índice tres veces
                if k == i or k == j:
                    continue
                
                # El último dígito debe ser par para que el número sea par
                if digits[k] % 2 == 0:
                    # Construir el número de 3 dígitos
                    num = digits[i] * 100 + digits[j] * 10 + digits[k]
                    result.add(num)
    
    # Convertir el conjunto a lista y ordenar
    return sorted(list(result))
```
