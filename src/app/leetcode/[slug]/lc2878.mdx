# Obtener el Tamaño de un DataFrame

```text
DataFrame players:
+-------------+--------+
| Column Name | Type   |
+-------------+--------+
| player_id   | int    |
| name        | object |
| age         | int    |
| position    | object |
| ...         | ...    |
+-------------+--------+
```

Escribe una solución para calcular y mostrar el **número de filas y columnas** de `players`.

Devuelve el resultado como un array: `[número de filas, número de columnas]`

## Ejemplo

```text
Input:
+-----------+----------+-----+-------------+--------------------+
| player_id | name     | age | position    | team               |
+-----------+----------+-----+-------------+--------------------+
| 846       | Mason    | 21  | Forward     | RealMadrid         |
| 749       | Riley    | 30  | Winger      | Barcelona          |
| 155       | Bob      | 28  | Striker     | ManchesterUnited   |
| 583       | Isabella | 32  | Goalkeeper  | Liverpool          |
| 388       | Zachary  | 24  | Midfielder  | BayernMunich       |
| 883       | Ava      | 23  | Defender    | Chelsea            |
| 355       | Violet   | 18  | Striker     | Juventus           |
| 247       | Thomas   | 27  | Striker     | ParisSaint-Germain |
| 761       | Jack     | 33  | Midfielder  | ManchesterCity     |
| 642       | Charlie  | 36  | Center-back | Arsenal            |
+-----------+----------+-----+-------------+--------------------+

Output:
[10, 5]

Explicación:
Este DataFrame contiene 10 filas y 5 columnas.
```

## Soluciones

<details>
    <summary>Python</summary>
```python
import pandas as pd

def getDataframeSize(players: pd.DataFrame) -> List[int]:
    rows, cols = players.shape

    return [rows, cols]
```
</details>
