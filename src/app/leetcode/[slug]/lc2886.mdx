# Cambiar Tipo de Datos

**DataFrame** `students`
<TerminalOutput>
{`+-------------+--------+
| Column Name | Type   |
+-------------+--------+
| student_id  | int    |
| name        | object |
| age         | int    |
| grade       | float  |
+-------------+--------+`}
</TerminalOutput>

Escribe una solución para corregir los errores:

- La columna `grade` está almacenada como flotantes, conviértela a enteros.

## Ejemplo

<MultiColumn>
<div>
**Input:**
<TerminalOutput>
{`DataFrame students:
+------------+------+-----+-------+
| student_id | name | age | grade |
+------------+------+-----+-------+
| 1          | Ava  | 6   | 73.0  |
| 2          | Kate | 15  | 87.0  |
+------------+------+-----+-------+`}
</TerminalOutput>
</div>
<div>
**Output:**
<TerminalOutput>
{`+------------+------+-----+-------+
| student_id | name | age | grade |
+------------+------+-----+-------+
| 1          | Ava  | 6   | 73    |
| 2          | Kate | 15  | 87    |
+------------+------+-----+-------+

Explicación: 
El tipo de datos de la columna "grade" 
se convierte a int.`}
</TerminalOutput>
</div>
</MultiColumn>

## Soluciones

<Solution title="Python" icon="Python">
```python
import pandas as pd

def changeDatatype(students: pd.DataFrame) -> pd.DataFrame:
    students['grade'] = students['grade'].astype(int)
    
    return students 
```
</Solution>
