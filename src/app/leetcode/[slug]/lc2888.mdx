**Input**

<TerminalOutput>
{`df1                                df2

| student_id | name    | age |     | student_id | name | age |

## Soluciones

<Solution title="Python" icon="Python">
```python
import pandas as pd

def concatenateTables(df1: pd.DataFrame, df2: pd.DataFrame) -> pd.DataFrame:
    return pd.concat([df1,df2])
```
