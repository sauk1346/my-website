Escribe una solución para **pivotear** los datos de modo que cada fila represente las temperaturas de un mes específico, y cada ciudad sea una columna separada.

## Soluciones

<Solution title="Python" icon="Python">
```python
import pandas as pd

def pivotTable(weather: pd.DataFrame) -> pd.DataFrame:
    return weather.pivot(
        index='month',
        columns='city',
        values='temperature'
    )
```
</Solution>
