# Encuentra el String Original Tecleado I

Alice está intentando escribir un string específico en su computadora. Sin embargo, tiende a ser torpe y **puede** presionar una tecla por demasiado tiempo, resultando en que un carácter sea tecleado múltiples veces.

Aunque Alice trató de concentrarse en su escritura, es consciente de que aún puede haber hecho esto **como máximo** *una vez*.

Se te da un string `word`, que representa la salida **final** mostrada en la pantalla de Alice.

Devuelve el número total de cadenas originales *posibles* que Alice *podría* haber intentado escribir.


## Ejemplos

**Ejemplo 1:**
> **Input:** word = "abbcccc"
> **Output:** 5
> **Explicación:**
> Las cadenas posibles son: "abbcccc", "abbccc", "abbcc", "abbc", y "abcccc".

**Ejemplo 2:**
> **Input:** word = "abcd"
> **Output:** 1
> **Explicación:**
> La única cadena posible es "abcd".

**Ejemplo 3:**
> **Input:** word = "aaaa"
> **Output:** 4

## Soluciones

<details>
    <summary>Python</summary>
```python
class Solution:
    def possibleStringCount(self, word: str) -> int:
        count = 1  # incluye la cadena original
        
        for i in range(1, len(word)):
            if word[i] == word[i-1]:
                count += 1
                
        return count
```
