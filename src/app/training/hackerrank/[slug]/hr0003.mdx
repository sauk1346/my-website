# "Hola, Mundo!" en C

Imprime `Hola, Mundo!` en una sola línea, y luego imprime la cadena de entrada ya proporcionada a stdout. Si no estás familiarizado con C, investiga sobre el comando `printf()`.

## Ejemplo

> **Input:**
> Bienvenido a programar en C.

> **Sample Output:**
> Hola, Mundo!
> Bienvenido a programar en C.

# Soluciones

<details>
    <summary>C</summary>
```c
#include <stdio.h>

int main() {

    char s[100];
    scanf("%[^\n]%*c", &s);
  	
    printf("Hola, Mundo!\n");
    printf("%s", s);    
    
    return 0;
}
```

- `scanf("%[^\n]%*c", &s)` lee toda la línea hasta el salto de línea
- `%[^\n]` lee todo excepto el salto de línea
- `%*c` consume el salto de línea sin guardarlo
</details>
