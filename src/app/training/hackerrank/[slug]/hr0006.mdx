# Fundamentos de Conjuntos y Relaciones

Sean 2 conjuntos:


- $Conjunto A = {1,2,3,4,5,6}$
- $Conjunto B = {2,3,4,5,6,7,8}$

¿Cuántos elementos están presente en $A \cup B$?

## Solución

<details>
    <summary>Matemáticas</summary>

- $A = \{1,2,3,4,5,6\}$
- $B = \{2,3,4,5,6,7,8\}$
- $A \cup B = \{1,2,3,4,5,6,7,8\}$

$\therefore$ hay 8 elementos en $A \cup B$ 
</details>
