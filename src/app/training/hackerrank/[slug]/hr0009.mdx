# Imprimiendo con Echo

Escribe un script que imprima la cadena "HELLO".


## Ejemplo

> **Output**
> HELLO

## Soluciones

<details>
    <summary>Bash</summary>
```bash
#!/bin/bash
echo "HELLO"
```

- `#!/bin/bash` : Llamado Shebang. Le dice al OS qué programa usar para ejecutar el script (ejecutar bash). "bin/bash" es la ubicación de bash en la mayoría de sistemas Unix/Linux.
- `echo` :  es un comando que imprime texto en la pantalla.

</details>
