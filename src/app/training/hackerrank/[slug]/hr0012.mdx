## Requisitos

1. Cuando la aplicación se carga, debe mostrar una lista vacía.
2. El campo de entrada debe aceptar texto del usuario.
3. Cuando se hace click en el botón:
    1. El texto del campo de entrada debe agregarse a la lista.
    2. Luego de hacer click, el campo de entrada debe limpiarse.
    3. Si el campo de entrada está vacío y se hace click, no se debe agregar nada a la lista.

## Ejemplo

**Estado Inicial:**
- La lista está vacía.
- El campo de entrada está vacío. 

**Acción 1:**
- Usuario escribe "Primer Elemento" en el campo de entrada.
- Usuario hace click en el botón "Agregar Elemento".
- La lista muestra "Primer Elemento".

**Acción 2:**
- Usuario escribe "Segundo Elemento" en el campo de entrada.
- Usuario hace click en el botón "Agregar Elemento".
- La lista muestra:
    - "Primer Elemento"
    - "Segundo Elemento"

1. `trim()` elimina espacios en blanco al inicio y final de la cadena
2. `if (input.trim() !== "")` validación de entrada. Verifica que después de quitar espacios en blanco, el texto no este vacío.
3. `setItems([...items, input.trim()]);` agrega elementos a la lista
4. `...items` spread operator. Toma todos los elementos actuales del array `items` y los expande a un nuevo array
5. `input.trim()` agrega el nuevo elemento (sin espacios extra) al final de nuevo array 

Es necesario crear un nuevo arreglo, ya que se utiliza "comparación por referencia" (no detecta cambios si se apunta al mismo arreglo)
