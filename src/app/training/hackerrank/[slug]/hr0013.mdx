# Coincidir una Cadena

Tienes una cadena de prueba $S$. Tu tarea es hacer coincidir la cadena `hackerrank`. Esto distingue entre mayúsculas y minúsculas.

## Soluciones

<details>
    <summary>Python</summary>
```python
import re

Regex_Pattern = r'hackerrank'

# código generado automático para revisión de regex
Test_String = input()
match = re.findall(Regex_Pattern, Test_String)
print("Number of matches :", len(match))
```
