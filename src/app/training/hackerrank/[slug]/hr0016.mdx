# Suma Simple de Array

Dado un array de enteros, encuentra la suma de sus elementos.

## Ejemplo

**Input:**
```text
STDIN           Function
-----           --------
6               ar[] size n = 6
1 2 3 4 10 11   ar = [1, 2, 3, 4, 10, 11]
```

**Output**
```text
31
```

## Soluciones

<details>
    <summary>Pseudocódigo</summary>

El algoritmo consiste en recorrer el array `A` y sumar todos sus elementos
- Se inicializa la variable `sum` en `0`
- Se realiza un loop `for` desde `i = 1` hasta `n` (cantidad de elementos de `A`)
- En cada iteración, se suma el elemento actual `A[i]` al acumulador `sum`
- Al finalizar el loop `for`, se devuelve el valor total acumulado en `sum`

```text
Algorithm: Suma Simple de Array
Input:     Array A de n enteros
Output:    Suma de elementos en A

sum ← 0
for i = 1 to n do
    sum ← sum + A[i]
end for
return sum
```
</details>

<details>
    <summary>C</summary>
```c
int simpleArraySum(int ar_count, int* ar) {
    int cont = 0;
    for (int i = 0; i < ar_count; i++) {
        cont = cont + ar[i];
    }
    return cont;
}
```
