# Jugando con Caracteres

Imprime un caracter en la primera línea.

Luego una palabra en la segunda línea.

En la última línea imprime una oración.

## Ejemplo

**Input**
> C
> Language
> Welcome To C!!

**Output**
> C
> Language
> Welcome To C!!

## Soluciones

<details>
    <summary>C</summary>
```c
#include <stdio.h>

int main() {
    
    char ch;
    char s[100];
    char sen[100];
    
    scanf("%c", &ch);
    scanf("%s", s);
    scanf(" %[^\n]", sen); 
    
    printf("%c\n", ch);
    printf("%s\n", s);
    printf("%s\n", sen);
        
    return 0;
}
```
