## Arreglo 2D

Dado un arreglo 2D `arr` de $6 \times 6$, un reloj de arena es un subconjunto de valores con índices que siguen el siguiente patrón:

```text
a b c
  d
e f g
```

Hay 16 relojes de arena en un arreglo de $6 \times 6$. La **suma del reloj de arena** es la suma de los valores que forman el reloj de arena.

Calcula la suma del reloj de arena para cada reloj en `arr`, luego imprime la **suma más alta** del reloj de arena encontrado.

## Ejemplo

```text
arr =  -9 -9 -9  1  1  1 
        0 -9  0  4  3  2
       -9 -9 -9  1  2  3
        0  0  8  6  6  0
        0  0  0 -2  0  0
        0  0  1  2  4  0
```

Las 16 sumas de los relojes de arena son:

```text
-63, -34, -9, 12, 
-10,   0, 28, 23, 
-27, -11, -2, 10, 
  9,  17, 25, 18
```

La suma más alta corresponde al reloj de arena encontrado en la fila 1, columna 2:

```text
0 4 3
  1
8 6 6
```

## Soluciones


<details>
    <summary>Pseudocódigo</summary>

- Se hace un recorrido horizontal y vertical para capturar las sumas de los 16 relojes de arenas.
- Esto se realiza mediante `FOR` anidados
    - el primer `FOR` representa el recorrido vertical
        - rango de iteración: `rango_vertical_arr - rango_vertical_reloj + 1` = `6 - 3 + 1` = `4` 
    - el segundo `FOR` representa el recorrido horizontal
        - rango de iteración: `rango_horizontal_arr - rango_horizontal_reloj + 1` = `6 - 3 + 1` = `4` 

- En cada recorrido compara la suma guardada, con la suma actual
    - si la suma actual es mayor que la guardada, reemplaza a la suma guardada
    - en caso contrario, mantiene el valor de la suma guardada     
- Al terminar las iteraciones, devuelve el valor de la suma guardada

```text
ALGORITHM SumaMaximaRelojArena
INPUT: A[6][6] ← arreglo bidimensional de enteros
OUTPUT: max_suma ← entero (suma máxima del reloj de arena)

BEGIN
    max_suma ← -∞
    
    FOR i ← 0 TO 3 DO                                                 // 0 a 3, son 4 iteraciones
        FOR j ← 0 TO 3 DO                                             // 0 a 3, son 4 iteraciones
            current_sum ← A[i][j] + A[i][j+1] + A[i][j+2] +           // fila superior
                                   A[i+1][j+1] +                      // centro
                          A[i+2][j] + A[i+2][j+1] + A[i+2][j+2]       // fila inferior
            
            IF current_sum > max_suma THEN
                max_suma ← current_sum
            END IF
        END FOR
    END FOR
    
    RETURN max_suma
END
```

```python
def hourglassSum(arr):
    
    max_sum = -99999
    sum_hourglass = 0
    for i in range(4):
        for j in range(4):
            sum_hourglass = arr[i][j] + arr[i][j+1] + arr[i][j+2] + arr[i+1][j+1] + arr[i+2][j]+ arr[i+2][j+1]+ arr[i+2][j+2]
            
            max_sum = sum_hourglass if sum_hourglass > max_sum else max_sum
    
    return max_sum 
```
