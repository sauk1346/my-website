# Fundamentos de Conjuntos y Relaciones \#2

Sean 2 conjuntos:

- $A = \{1, 2, 3, 4, 5, 6\}$
- $B = \{2, 3, 4, 5, 6, 7, 8\}$

¿Cuántos elementos hay en $A \cap B$ ?

## Soluciones

<details>
    <summary>Matemáticas</summary>

- $A = \{1, 2, 3, 4, 5, 6\}$
- $B = \{2, 3, 4, 5, 6, 7, 8\}$
- $A \cap B = \{2, 3, 4, 5, 6\}$

$\therefore$ hay 5 elementos en $A \cap B$
</details>
