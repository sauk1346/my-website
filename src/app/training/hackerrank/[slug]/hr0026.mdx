# If-Else

Dado un entero `n`, realiza las siguientes acciones condicionales:
- Si $n$ es impar, imprime `Weird`
- Si $n$ es par y está en el rango inclusivo de 2 a 5, imprime `Not Weird`
- Si $n$ es par y está en el rango inclusivo de 6 a 20, imprime `Weird`
- Si $n$ es par y mayor que 20, imprime `Not Weird`

# Ejemplo

> **Input** 
> 3

> **Output**
> Weird

## Soluciones

def classify_number(n):
    
    # Verificar si n es impar
    if n % 2 == 1:
        return "Weird"
    
    # Si n es par, verificar los rangos
    else:
        if 2 <= n <= 5:
            return "Not Weird"
        elif 6 <= n <= 20:
            return "Weird"
        else:  
            return "Not Weird"
