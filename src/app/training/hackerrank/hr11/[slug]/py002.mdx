# Python If-Else

## Task
Given an integer $n$, perform the following conditional actions:

- If $n$ is odd, print `Weird`
- If $n$ is even and in the inclusive range of $2$ to $5$, print `Not Weird`
- If $n$ is even and in the inclusive range of $6$ to $20$, print `Weird`
- If $n$ is even and greater than $20$, print `Not Weird`

### Input Format
A single line containint a positive integer, $n$

### Constraints
- `1 <= n <= 100`

### Ouput Format
Print `Weird` if the number if weird. Otherwise, print `Not Weird`

## Examples

**Example 1:**
> **Input:** 3
> **Output:** Weird

**Example 2:**
> **Input:** 24 
> **Output:** Not Weird

#odd = impar

#1. n es impar                         : imprimir "Weird"
#2. n es par y esta en el rango 2 a 5  : imprimir "Not Weird"
#3. n es par y esta en el rango 6 a 20 : imprimir "Weird"
#4. n es par y es mayor a 20           : imprimir "Not Weird"

if __name__ == '__main__':
    n = int(input().strip())

    if n%2 != 0:
        print("Weird")        
    elif n%2 == 0 and n in range(2,6): 
        print("Not Weird")
    elif n%2 == 0 and n in range(6,21):
        print("Weird")
    elif n%2 == 0 and n>20:
        print("Not Weird")
