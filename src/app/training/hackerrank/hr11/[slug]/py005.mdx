# Loops

## Task 
The provided code stub reads and integer, $n$, from STDIN. For all non-negative integers $i < n$, print $i^2$

### Input Format
The first and only line contrains the integer, $n$.

### Constraints
- $1 \le n \le 20$

### Output Format
Print $n$ lines, one corresponding to each $i$.

### Example

> **Input**
> 5

> **Output**
> 0
> 1
> 4
> 9
> 16

<details>
    <summary>Solution</summary>
```python
if __name__ == '__main__':
    n = int(input())
    
    for i in range(n):
        print(i*i)
``` 
</details>
