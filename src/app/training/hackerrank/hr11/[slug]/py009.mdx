# Find the Runner-up Score!

Given the participants' score sheet for your University Sport Day, you are required to find the runner-up score. You are given $n$ scores. Store them in a list and find the score of the runner-up.

## Example

> **Input**
> 5
> 2 3 6 6 5

> **Output**
> 5

**Explanation:** Given list is $[2,3,6,6,5]$. The maximun score is 6. second maximun is 5. Hence, we print 5 as the runner-up score.

<details>
    <summary>Solution</summary>
```python
if __name__ == '__main__':
    n = int(input())
    arr = map(int, input().split())

    unique_scores = sorted(set(arr), reverse=True)
    runnerUp = unique_scores[1]
    print(runnerUp)
```
