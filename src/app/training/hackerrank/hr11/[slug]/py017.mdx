# Mutations

We have seen that list are mutable (they can be changed), and tuples are immutable (they cannot be changed).

## Task
Read a given string, change the character at a given index and then print the modified string.

## Example

**Input**
abracadabra
5 k

**Output**
abrackadabra

<details>
    <summary>Solution</summary>
```python
def mutate_string(string, position, character):
    my_list = list(string)
    my_list[position] = character
    string = ''.join(my_list)
    return string
```
</details>
