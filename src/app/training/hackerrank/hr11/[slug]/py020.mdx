# Text Alignment

In Python, a string of text can be aligned left, right and center.

1. `.ljust(widh)` : Returns a left aligned string of length width

```python
>>> width = 20
>>> print 'HackerRank'.ljust(width,'-')
HackerRank----------  
``` 

2. `.center(width)` : Returns a centered string of length width

```python
>>> width = 20
>>> print 'HackerRank'.center(width,'-')
-----HackerRank-----
```

3. `.rjust(width)` : Return a right alined string of length width

```python
>>> width = 20
>>> print 'HackerRank'.rjust(width,'-')
----------HackerRank
```
