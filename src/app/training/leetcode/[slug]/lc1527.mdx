# Pacientes con una Condición

**Tabla:** `Patients`

```markdown
+--------------+---------+
| Column Name  | Type    |
+--------------+---------+
| patient_id   | int     |
| patient_name | varchar |
| conditions   | varchar |
+--------------+---------+

- "patient_id" es la clave primaria (columna con valores únicos)
   para esta tabla.

- "conditions" contiene 0 o más códigos separados por espacios.
  Esta tabla contiene información de los pacientes en el hospital.
```

Escriba una solución para encontrar `patient_id`, `patient_name` y `conditions` de los pacientes que tienen "Diabetes Tipo I". `Type I Diabetes` siempre comienza con el prefijo `DIAB1`.

Retorne la tabla de resultados en **cualquier orden**.

## Ejemplo

**Input:**
```markdown
Patients table:
+------------+--------------+--------------+
| patient_id | patient_name | conditions   |
+------------+--------------+--------------+
| 1          | Daniel       | YFEV COUGH   |
| 2          | Alice        |              |
| 3          | Bob          | DIAB100 MYOP |
| 4          | George       | ACNE DIAB100 |
| 5          | Alain        | DIAB201      |
+------------+--------------+--------------+
``` 

**Output:** 
```markdown
+------------+--------------+--------------+
| patient_id | patient_name | conditions   |
+------------+--------------+--------------+
| 3          | Bob          | DIAB100 MYOP |
| 4          | George       | ACNE DIAB100 | 
+------------+--------------+--------------+
```

**Explicación:**
> Bob and George ambos tienen una condición que comienza con DIAB1.

## Solución

<details>
    <summary>Python</summary>
```python
def find_patients(patients: pd.DataFrame) -> pd.DataFrame:
    
    regex = r' DIAB1.*|^DIAB1.*'

    return patients[patients['conditions'].str.contains(regex, na=False)]
```
</details>
