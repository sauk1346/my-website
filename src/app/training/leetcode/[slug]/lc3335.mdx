# Total Characters in String After Transformations I

You are given a string `s`and an integer `t`, representing the number of **transformations** to perform. In one **transformation**, every character in `s`is replaces according to the following rules:

- If the character is `z`, replace it with the string `"ab"`
- Otherwise, replace it with the **next** character in the alphabet. For example, `'a'` is replaced with `'b'`, `'b'` is replaced with `'c'` and so on.

Return the **length** of the resulting string after **exactly** `t` transformations.

Since the answer may be very larte, return it **modulo** `10⁹ + 7`

## Examples

### Example 1

> **Input:** s = "abcyy", t = 2
> **Output:** 7

**Explanation:**
- **First Transformation (t=1)**:
    - 'a' becomes 'b'
    - 'b' becomes 'c'
    - 'c' becomes 'd'
    - 'y' becomes 'z'
    - 'y' becomes 'z'
    - String after the first transformation: "bcdzz"

- **Second Transformation (t = 2):**
    - 'b' becomes 'c'
    - 'c' becomes 'd'
    - 'd' becomes 'e'
    - 'z' becomes "ab"
    - 'z' becomes "ab"
    - String after the second transformation: "cdeabab"

- **Final Length of the string:** The string is "cdeabab", which has 7 characters.


### Example 2

> **Input:** s = "azbk", t = 1
> **Output:** 5

**Explanation:**
- **First Transformation (t = 1)**:
    - 'a' becomes 'b'
    - 'z' becomes "ab"
    - 'b' becomes 'c'
    - 'k' becomes 'l'
    - String after the first transformation: "babcl"

- **Final Length of the string:** The string is "babcl", which has 5 characters.

## Solutions

<details>
    <summary>Python</summary>
```python
class Solution:
    def lengthAfterTransformations(self, s: str, t: int) -> int:
        MOD = 10**9 + 7
        
        current_s = list(s)
        for _ in range(t):
            new_s = []
            for letter in current_s:
                if letter == 'z':
                    new_s.append('a')
                    new_s.append('b')
                else:
                    new_s.append(chr(ord(letter) + 1))
            
            current_s = new_s
        
        return len(current_s) % MOD
```

- Se crea una lista `current_s` a partir del string `s`
- Se crea una lista vacía  `new_s` para realizar los cambios
- Se recorre la lista `current_s` y se guardan los cambios en `new_s`
- Se actualiza la lista `current_s` con los cambios de `new_s` por cada transformación completada
- Se retorna el largo de `current_s`, acotado por MOD
