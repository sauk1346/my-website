# Fundamentos de Git y GitHub

- Git es un sistema de control de versiones distribuido que te permite rastrear cambios en tu código y colaborar con otros desarrolladores.
- Fue diseñado por Linux Torvalds.

## 3. Creación de un repositorio

1. **Crear nuevo repositorio:** Buscar el botón "new". Alternativamente se puede ir al siguiente link
    ```text
    https://github.com/new
    ```

2. **Nombre repositorio:** Crear un nombre sin espacios. Luego botón "Crear"

3. **Configuración terminal:** Crear un directorio desde la terminal
    ```sh
    mkdir miproyecto
    cd miproyecto
    ```

4. **Inicializar repositorio local:** 
    ```sh
    git init
    ```
