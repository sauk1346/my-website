## 1. DuckDB Shell

Es la interfaz de línea de comandos (CLI) de DuckDB, un sistema de gestión de bases de datos analíticas embebido y de alto rendimiento.

### Ejercicio

    ```sql
    select * from

    09-03/stackoverflow_survey_questions.csv";
    ```
