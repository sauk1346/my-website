## 2. Machine Learning with Python
- Explore algorithms - linear and logistic regression
- Learn about decision trees and build predictive models

## 3. Deep learning & neural network with Keras
- Learn about unsupervised learning
- Explain convolutional and recurrent networks
- Explore autoencoders and restricted Boltzmann machines
- Build deep learning models and networks

## 4. Deep learning with Keras and TensorFlow
- Create custom layers and models and integrate Python libraries
- Develop CNNs and transformer models
- Learn about unsupervised learning for model optimization
- Explore generative modeling and reinformcement learning principles

## 5. Neural networks and PyTorch
- Explore 2D tensors, derivatives, linear regression, and los calculation
- Manage batch processing, gradient descent, and logistic regression
- Build models for image recognition and predictive analytics

## 6. Deep learning with PyTorch
- Apply Softmax regression, multi-class neural networks, CNNs
- Explore overfitting, backpropagation, vanishing gradient
- Apply sigmoid, tanh, ReLU

## 7. AI capstone project with deep learning
- Apply deep learning to real-world problems
- Task:
    - Choose a library
    - Preprocess data
    - Build and test a model
    - Validate its performance
- Showcase model's efectiveness

## 8. Generative AI and LLMs
- Apply RNNs, transformers, GANs, VAEs, and diffusion models
- Explore training methods, tokenization techniques, data preparation processes
- Work with LLMs tokenizers, and data loaders
- Implement generative AI with Hugging Face

## 9. Generative AI: Foundational models for NLP
- Explore one-hot encoding, embedding, and Word2vec
- Build neural networks using PyTorch and Torch Text
- Apply N-gram and sequence-to-sequence models
- Evaluate text quality with BLEU metrics

## 10. Generative AI: Language modeling with transformers
- Experience transformer-based models
- Explore multi-head attention and decoder-based modeling
- Explore BERT's bidirectional encoding with MLM and NSP
- Implement PyTorch for language tasks

## 11. Generative AI: Engineering and fine-tuning transformers
- Use Hugging Face and PyTorch frameworks
- Optimize LLMs with PEFT, LoRA, QLoRA, and prompting
- Load, pre-train, and fine-tune models

## 12. Advanced fine-tuning for LLMs
- Learn about human feedback and direct preference
- Explore instruction-tuning and reinforcement learning techniques
- Apply reward modeling, PRO, and DPO to optimize LLMs

## 13. Fundamentals of AI agents using RAG and LangChain
- Explore RAG, prompt engineering, and LangChain
- Learn external data and tokenizers integration with RAG
- Use LangChain tools, components, and LLMs
- Apply skills in labs and projects for practical experience

## 14. Generative AI application with RAG and LangChain
- Apply LangChain document loader, text-splitting strategies
- Use watsonx and vector databases
- Develop QA bot and Gradio interface
