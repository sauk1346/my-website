# Introducción a Variables y Comentarios

## 1. Comentarios
Son líneas de código que el intérprete de JavaScript ignorará a la hora de ejecutarlo

```js
// esta línea esta comentada
console.log("Hola mundo!")
```

## 2. Variables

```js
let a = 10;
var b = 10;
const c = 10;
```

- `let`: Declara una variable con alcance de bloque, reasignable pero no redeclarable en el mismo scope.
- `var`: Declara una variable con alcance de función, reasignable y redeclarable (legacy, evitar en código moderno).
- `const`: Declara una constante con alcance de bloque, no reasignable ni redeclarable (el valor/referencia es inmutable).

## 2. Depuración y breakpoints

1. Ir a consola de 'google chrome'
2. Pestaña 'Sources'. Muestra `index.html` y `app.js`

### Breakpoint
- Es posible seleccionar una línea de código, y establecer un breakpoint.
- Esto hará que la ejecución se detenga al llegar a dicha línea
