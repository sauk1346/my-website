# Configuración Godot

1. **Crear proyecto:** 'platzipong'
2. **Configurar:**
    - General -> Display -> Window:
        - Width: 1024
        - Height: 1080
    - Mode: viewport
    - Aspect: keep

## 2. Arbol de escenas

- Es posible cargar y crear todos los nodos que queramos
- Pero es recomendable organizar todo en escenas
- Cada escena puede tener su propio árbol de nodos

## 3. Panel de importación
- Permite configurar cómo se importan los recursos al proyecto
- Por ejemplo, las imágenes pueden ser importadas como texturas o sprites
- Es importante revisar este panel para asegurarse de que los recursos se importen correctamente
