## 1. ¿Qué es?

Permite tomar los hallazgos encontrados de diferentes problemáticas mediante la observación, las entrevistas, las fichas de antecedentes, etc., para resignificar las problemáticas del desafío planteado.

## 2. ¿De qué se trata?

Utilizar:

- Procesamiento convergente y divergente
- Colocar hallazgos y problemáticas
- Redefinición por temáticas

Esto permite definir problemáticas más complejas de resolver y las de mayor impacto.

## 3. ¿Para qué sirve?

- Permite identificar probemas reales
- Darles un significado
- Filtrar y priorizar las problemáticas 

## 4. ¿Cómo se utiliza?

1. Matriz de vaciado de hallazgos (brainstorming)
2. Agrupación (insights)
3. Clasificación mediante puntuación
    - Problema más complejo a resolver y con mayor impacto al usuario objetivo

## 5. ¿Qué necesita?

- Información de las fases anteriores
- Actividades para abrir la mente y detectar problemas reales
- Pensamiento crítico para discenir

## 6. ¿Qué aporta?

- Resignar problemáticas iniciales
- Encontrar problemas reales
- Encontrar soluciones innovadoras y útiles
