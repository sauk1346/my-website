**Paso 5: Configurar `Switch0`**

<TerminalOutput withBg>
{`# entrar al modo privilegiado
enable

# entrar al modo de configuración global
configure terminal

# entrar a la interfaz virtual de administración
interface vlan 1

# asignar IP y máscara a la interfaz VLAN 1
ip address 172.16.0.2 255.255.240.0

# activar la interfaz
no shutdown

# salir de la interfaz
exit

# definir la puerta de enlace
ip default-gateway 172.16.0.1`}
</TerminalOutput>

**Paso 7: Configurar Router `STGO-CENTRO`**

<TerminalOutput withBg>
{`# entrar al modo privilegiado
enable

# entrar al modo de configuración global
configure terminal

# 1. asignar nombre al router
hostname STGO-CENTRO

# 2. configuración interfaz G0/0 (red izquierda)
interface GigabitEthernet0/0
ip address 172.16.0.1 255.255.240.0
# activar la interfaz
no shutdown
# salir de la interfaz
exit

# 3. configuración interfaz G0/1 (red derecha)
interface GigabitEthernet0/1
ip address 172.16.16.1 255.255.240.0
# activar la interfaz
no shutdown
# salir de la interfaz
exit`}
</TerminalOutput>
