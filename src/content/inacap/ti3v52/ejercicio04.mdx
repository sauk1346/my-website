3. Configuraciones básicas y de seguridad SSH en el router:
    - Asigne su propio nombre (en Mayúscula) como nombre al router, ejemplo: JUAN
    - Asigne su propio apellido.cl como nombre de dominio para SSH, ejemplo: perez.cl
    - Crear usuario con nombre: admin3 contraseña cifrada: cisco-e3
    - Contraseña cifrada para modo privilegiado: cisco-e3
    - Configurar clave RSA con largo de la clave 1024 bits
    - Habilite versión 2 de SSH
    - Tiempo máximo de inactividad 60 seg
    - Intentos de autenticación: 2
    - Habilite ssh en líneas vty
    - Al conectarse por SSH debe solicitar usuario y contraseña
    - Desde PC0, en Command Promtp, ejecute comando:
 `ssh - l admin3 192.168.X.1`
    - Ingrese la contraseña de admin3, debe aparecer el nombre del router.

<Solution title="Solución Pregunta 3">

<TerminalOutput 
    withBg
    caption="Configuraciones básicas y de seguridad SSH en Router0"
>
{`# entrar al modo privilegiado
enable

# entrar al modo de configuración global
configure terminal

# 1. asignar nombre al router
hostname ANTONIO

# 2. asignar nombre de dominio para SSH
ip domain-name morales.cl

# 3. crear usuario con contraseña cifrada
username admin3 secret cisco-e3

# 4. contraseña cifrada para modo privilegiado
enable secret cisco-e3

# 5. generar claves RSA de 1024 bits para SSH
crypto key generate rsa modulus 1024

# 6. habilitar versión 2 de SSH
ip ssh version 2

# 7. tiempo máximo de inactividad: 60 segundos
ip ssh time-out 60

# 8. intentos máximos de autenticación
ip ssh authentication-retries 2

# 9. habilitar SSH en líneas vty y solicitar usuario/contraseña
line vty 0 4
# deshabilitar todos los protocolos de acceso remoto
no transport input
# permitir solo SSH
transport input ssh
login local
# salir de las líneas vty
exit

# salir del modo de configuración global
exit`}
</TerminalOutput>
