# Estándares IEEE 802.3 y IEEE 802.11

- **Estándar 802.3** $\rightarrow$ Ethernet (Redes cableadas)
- **Estándar 802.11** $\rightarrow$ Wi-Fi (Redes inalámbricas)

Estos actúan como un idioma común, asegurando la interoperabilidad de dispositivos de distintos fabricantes en cualquier parte del mundo.

## 1. El Comité IEE 802 y las Capas OSI

El comité IEEE 802 define un conjunto de estándares que rigen la forma en que se envían los datos y cómo los dispositivos obtienen acceso al medio.

Los estándares 802 se relacionan directamente con las dos capas inferiores del Modelo de Referencia OSI (Cisco Networking Academy, 2015):

## 2. Estándar IEEE 802.3 - Ethernet (redes cableadas)

Ethernet es la tecnología predominante en el mundo para redes que utilizan cableado.

### Mecanismo de acceso y componentes

El estándar IEEE 802.3 define el mecanismo CSMA/CD (Acceso Múltiple con Detección de Colisiones) como la forma fundamental para que los dispositivos accedan al medio compartido. (Cisco Networking Academy, 2015)

## 3. Estándar IEEE 802.11 - WiFi (redes inalámbricas)

El 802.11 define cómo viajan los datos a través del aire mediante ondas de radio.

Los dispositivos Wi-Fi utilizan el mecanismo CSMA/CA (Acceso Múltiple con Evasión de Colisiones) para turnarse y acceder al medio compartido del aire.

### Frecuencias y evolución

Se observan los siguientes datos:

- **2.4 GHz:** mayor alcance, menor velocidad y más interferencias
- **5 GHz:** menor alcance, mayor velocidad y menos interferencias

### Dispositivo Principal: Punto de Acceso (AP)

El Access Point (AP) conecta clientes inalámbricos (802.11) a la red cableada (802.3) y realiza la conversión de las tramas entre ambos estándares.

### Relación Integrada con OSI/TCP-IP

La transmisión de datos implica la encapsulación desde Capa 7 (Aplicación) hasta Capa 2 (Trama Ethernet o Wi-Fi) utilizando las especificaciones de Capa 1 para el envío físico. El direccionamiento IP ocurre en Capa 3.
