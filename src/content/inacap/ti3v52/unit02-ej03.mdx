    - **red LAN 4:** contiene al host PC6 y PC7

<P indent={2}>
**Paso 02: Configuración interfaces routers ENTEL y VTR**

Por convención, cada interfaz tiene la 1ra IP de la red anexa. Entonces:
- **Interfaz g0/0 (ENTEL):** 1ra IP de red LAN 1
- **Interfaz g0/1 (ENTEL):** 1ra IP de red LAN 2
- **Interfaz s0/0/0 (ENTEL):** 1ra IP de red WAN 1
- **Interfaz g0/0 (VTR):** 1ra IP de red LAN 3
- **Interfaz g0/1 (VTR):** 1ra IP de red LAN 4
- **Interfaz s0/0/1 (VTR):** 2da IP de red WAN 1
</P>

<P indent={2}>
**Paso 04: Rutas estáticas**

- Se crean tablas de rutas estáticas, para que los routers puedan *ver* las redes más lejanas
- Router ENTEL solo ve LAN 1 y LAN 2. Con ruta estática podrá ver las redes LAN 3 y LAN 4
- Router VTR solo ve LAN 3 y LAN 4. Con ruta estática podrá ver las redes LAN 1 y LAN 2
</P>
