# Fundamentos de cloud computing: la nube actual

El *cloud computing* es un modelo tecnológico que permite el acceso remoto, bajo demanda y a través de internet, a recursos informáticos como servidores, almacenamiento, redes y aplicaciones. Este paradigma se basa en la virtualización y la distribución de recursos, ofreciendo **escalabilidad**, **flexibilidad** y **reducción de costos** operativos.

## 1. Modelos de servicio principales

El cloud computing se organiza en tres modelos de servicio fundamentales, cada uno definiendo el nivel de control y responsabilidad que tiene el usuario sobre la infraestructura.

### A. IaaS (Infraestructure as a Service)

Proporciona acceso a infraestructura virtualizada como servidores, redes y almacenamiento. El usuario gestiona el sistema operativo, middleware y aplicaciones, mientras el proveedor administra el hardware físico. Es el modelo más flexible y de mayor control para el usuario.

> **Ejemplos:** Amazon EC2, Google Compute Engine, Azure Virtual Machines.

### B. PaaS (Platform as a Service)

Ofrece una plataforma completa de desarrollo y despliegue en la nube. El usuario se enfoca exclusivamente en desarrollar y desplegar aplicaciones, sin preocuparse por la infraestructura subyacente ni el sistema operativo.

> **Ejemplos:** Google App Engine, Azure App Service, Heroku.

### C. SaaS (Software as a Service)

Es el modelo más abstracto. El usuario consume directamente una aplicación lista para usar a través del navegador o cliente, sin gestionar ninguna capa tecnológica subyacente.

> **Ejemplos:** Gmail, Microsoft 365, Salesforce, Zoom.
