# Cantidad de Clientes Ricos

**Tabla** `Store`
<TerminalOutput>
{`+-------------+------+
| Column Name | Type |
+-------------+------+
| bill_id     | int  |
| customer_id | int  |
| amount      | int  |
+-------------+------+

- "bill_id" es la clave primaria para esta tabla.
- Cada fila contiene información sobre el monto de una factura y el cliente
    asociado.`}
</TerminalOutput>

Escribe una solución para reportar el número de clientes que tuvieron **al menos** una factura con un monto **estrictamente mayor** que `500`.

## Ejemplo

**Input:**
<TerminalOutput>
{`Tabla Store:
+---------+-------------+--------+
| bill_id | customer_id | amount |
+---------+-------------+--------+
| 6       | 1           | 549    |
| 8       | 1           | 834    |
| 4       | 2           | 394    |
| 11      | 3           | 657    |
| 13      | 3           | 257    |
+---------+-------------+--------+`}
</TerminalOutput>

**Output:**
<TerminalOutput>
{`+------------+
| rich_count |
+------------+
| 2          |
+------------+

Explicación:
- cliente1 tiene dos facturas con montos estrictamente mayores que 500.
- cliente2 no tiene ninguna factura con montos estrictamente mayores que 500.
- cliente3 tiene una factura con un monto estrictamente mayor que 500.`}
</TerminalOutput>

## Soluciones

<Solution title="Python" icon="Python">
```python
import pandas as pd

def count_rich_customers(store: pd.DataFrame) -> pd.DataFrame:
    
    rich_customers = store[store['amount'] > 500]['customer_id'].nunique()
    return pd.DataFrame({'rich_count': [rich_customers]})
```
</Solution>
