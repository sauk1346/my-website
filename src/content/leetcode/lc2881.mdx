# Crear una Nueva Columna

**DataFrame** `employees`
<TerminalOutput>
{`+-------------+--------+
| Column Name | Type.  |
+-------------+--------+
| name        | object |
| salary      | int.   |
+-------------+--------+`}
</TerminalOutput>


Una empresa planea proporcionar a sus empleados un bono.

Escribe una solución para crear una nueva columna llamada `bonus` que contenga los **valores duplicados** de la columna `salary`.

## Ejemplo

Input:
<TerminalOutput>
{`DataFrame employees
+---------+--------+
| name    | salary |
+---------+--------+
| Piper   | 4548   |
| Grace   | 28150  |
| Georgia | 1103   |
| Willow  | 6593   |
| Finn    | 74576  |
| Thomas  | 24433  |
+---------+--------+`}
</TerminalOutput>

Output:
<TerminalOutput>
{`+---------+--------+--------+
| name    | salary | bonus  |
+---------+--------+--------+
| Piper   | 4548   | 9096   |
| Grace   | 28150  | 56300  |
| Georgia | 1103   | 2206   |
| Willow  | 6593   | 13186  |
| Finn    | 74576  | 149152 |
| Thomas  | 24433  | 48866  |
+---------+--------+--------+

Explicación: 
Se crea una nueva columna bonus duplicando el valor de la columna salary.`}
</TerminalOutput>

## Soluciones

<Solution title="Python" icon="Python">
```python
import pandas as pd

def createBonusColumn(employees: pd.DataFrame) -> pd.DataFrame:
    employees['bonus'] = employees['salary'] * 2

    return employees
```
</Solution>
