# Eliminar Filas Duplicadas

**DataFrame** `customers`
<TerminalOutput>
{`+-------------+--------+
| Column Name | Type   |
+-------------+--------+
| customer_id | int    |
| name        | object |
| email       | object |
+-------------+--------+`}
</TerminalOutput>

Hay algunas filas duplicadas en el DataFrame basadas en la columna `email`.

Escribe una solución para eliminar estas filas duplicadas y mantener solo la **primera** ocurrencia.

## Ejemplo

**Input**
<TerminalOutput>
{`+-------------+---------+---------------------+
| customer_id | name    | email               |
+-------------+---------+---------------------+
| 1           | Ella    | emily@example.com   |
| 2           | David   | michael@example.com |
| 3           | Zachary | sarah@example.com   |
| 4           | Alice   | john@example.com    |
| 5           | Finn    | john@example.com    |
| 6           | Violet  | alice@example.com   |
+-------------+---------+---------------------+`}
</TerminalOutput>

**Output**  
<TerminalOutput>
{`+-------------+---------+---------------------+
| customer_id | name    | email               |
+-------------+---------+---------------------+
| 1           | Ella    | emily@example.com   |
| 2           | David   | michael@example.com |
| 3           | Zachary | sarah@example.com   |
| 4           | Alice   | john@example.com    |
| 6           | Violet  | alice@example.com   |
+-------------+---------+---------------------+

Explicación:
Alice (customer_id = 4) y Finn (customer_id = 5) ambos usan john@example.com, 
por lo que solo se conserva la primera ocurrencia de este email.`}
</TerminalOutput>

## Soluciones

<Solution title="Python" icon="Python">
```python
import pandas as pd

def dropDuplicateEmails(customers: pd.DataFrame) -> pd.DataFrame:
    return customers.drop_duplicates(subset=['email'], keep='first')
```
</Solution>
